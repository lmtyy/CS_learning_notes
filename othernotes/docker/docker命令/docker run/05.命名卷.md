好的！创建命名卷非常简单，主要有两种方式。

---

## 方式一：自动创建（最常用）

直接在 `docker run` 使用 `-v` 参数时指定卷名，Docker 会自动创建不存在的卷：

```bash
# 运行容器时自动创建命名卷
docker run -v 卷名:/容器内路径 镜像名
```

### 示例
```bash
# 自动创建名为 mysql_data 的卷并挂载
docker run -d --name my-mysql \
  -v mysql_data:/var/lib/mysql \
  -e MYSQL_ROOT_PASSWORD=123456 mysql

# 自动创建名为 app_data 的卷并挂载
docker run -v app_data:/app/data nginx
```

---

## 方式二：手动创建

使用 `docker volume create` 命令提前创建：

```bash
# 手动创建命名卷
docker volume create 卷名
```

### 示例
```bash
# 创建名为 my_volume 的卷
docker volume create my_volume

# 然后使用这个卷
docker run -v my_volume:/app/data nginx
```

---

## 查看和管理命名卷

### 查看所有卷
```bash
docker volume ls
```

### 查看卷详情
```bash
docker volume inspect 卷名
```

### 删除卷
```bash
# 删除单个卷
docker volume rm 卷名

# 删除所有未使用的卷
docker volume prune
```

---

## 完整示例

```bash
# 1. 创建命名卷
docker volume create my_app_data

# 2. 查看卷
docker volume ls

# 3. 使用卷运行容器
docker run -d --name my-app \
  -v my_app_data:/app/data \
  nginx

# 4. 查看卷的详细信息
docker volume inspect my_app_data
```

---

## 总结

创建命名卷就两种方法：
1. **`docker run -v 卷名:/路径`** - 自动创建（推荐）
2. **`docker volume create 卷名`** - 手动创建

**最简单常用的就是第一种**：直接在 `docker run` 时指定卷名，让 Docker 自动创建！
