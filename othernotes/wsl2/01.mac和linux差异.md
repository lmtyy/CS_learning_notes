很高兴你对macOS命令行熟悉，这让你能快速上手Linux（WSL2）。我来帮你梳理关键区别和上手步骤：

## 🚀 **核心差异速览**

| 方面 | macOS (BSD系) | Linux (GNU系, WSL2) | WSL2对应命令 |
|------|--------------|-------------------|-------------|
| **包管理器** | Homebrew (`brew`) | APT (`apt`) / 或选`brew` | `sudo apt update && sudo apt install` |
| **路径补全** | Tab补全（需设置） | 默认更智能的Tab补全 | 直接Tab，更友好 |
| **默认Shell** | zsh (新版本) | bash (默认) | `echo $SHELL` 查看 |
| **文本处理** | `sed -i ''` 需要空字符串 | `sed -i` 直接使用 | `sed -i 's/old/new/g' file` |
| **系统信息** | `system_profiler` | `uname -a`, `lsb_release -a` | `cat /etc/os-release` |

## 🔧 **WSL2环境快速配置**

```bash
# 1. 首先更新包列表
sudo apt update && sudo apt upgrade -y

# 2. 安装开发基础工具
sudo apt install -y build-essential git curl wget vim htop

# 3. 安装Python环境（AI科研必须）
sudo apt install -y python3 python3-pip python3-venv
pip3 install --upgrade pip

# 4. 配置CUDA环境（针对你的4060）
# 先检查WSL2 CUDA支持（需要Windows安装NVIDIA驱动）
nvidia-smi  # 应该能看到你的4060信息

# 5. 安装conda/miniconda（推荐）
wget https://repo.anaconda.com/miniconda/Miniconda3-latest-Linux-x86_64.sh
bash Miniconda3-latest-Linux-x86_64.sh
```

## 📁 **文件系统关键差异**

```bash
# 访问Windows文件（重要！）
cd /mnt/c/Users/你的用户名/  # C盘用户目录
cd /mnt/d/  # D盘（如果有）

# Linux家目录
cd ~  # 相当于 /home/你的用户名/

# 路径表示
# macOS/Linux通用: /home/user (绝对路径), ~/doc (相对家目录)
# 注意：Linux没有 /Volumes/，WSL2用 /mnt/
```

## 🐧 **常用命令对比表**

| 用途 | macOS示例 | Linux (WSL2) 示例 | 说明 |
|------|----------|------------------|------|
| **查看IP** | `ifconfig` 或 `ipconfig getifaddr en0` | `ip addr show` 或 `hostname -I` | Linux优先用`ip`命令 |
| **进程查看** | `ps aux` | `ps aux` (相同) | 或 `htop` (需安装) |
| **磁盘空间** | `df -h` | `df -h` (相同) | 加上 `du -sh .` 看当前目录大小 |
| **压缩解压** | `tar -zxf` | `tar -xzf` (相同) | Linux对参数顺序更灵活 |
| **环境变量** | `echo $PATH` | `echo $PATH` (相同) | 编辑用 `vim ~/.bashrc` |

## ⚡ **快速上手脚本**

创建 `~/.bash_aliases` 文件，添加以下别名：

```bash
# 常用别名
alias ll='ls -laFh'
alias cls='clear'
alias py='python3'
alias pip='pip3'

# 快速跳转到Windows目录
alias winhome='cd /mnt/c/Users/你的用户名/'
alias code='cd /mnt/c/Users/你的用户名/code'

# 查看GPU状态
alias gpu='nvidia-smi'
alias gpulight='nvidia-smi --query-gpu=utilization.gpu,memory.used --format=csv'

# Python虚拟环境快速创建
alias venv='python3 -m venv .venv && source .venv/bin/activate'
```

## 🐍 **AI环境配置实例**

```bash
# 创建AI项目环境
mkdir ~/ai-projects && cd ~/ai-projects
python3 -m venv pytorch-env
source pytorch-env/bin/activate

# 安装PyTorch（CUDA版本）
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121

# 验证安装
python -c "import torch; print(f'PyTorch版本: {torch.__version__}'); print(f'CUDA可用: {torch.cuda.is_available()}'); print(f'GPU: {torch.cuda.get_device_name(0)}')"
```

## 🚨 **常见问题解决**

1. **权限问题**：WSL2默认没有sudo密码，第一次用`sudo`会提示设置
2. **中文乱码**：`sudo apt install fonts-noto-cjk locales`
3. **GPU不可用**：确保Windows已装NVIDIA驱动，WSL2内核支持CUDA
4. **性能优化**：将项目文件放在WSL2内部（`~/`），而不是`/mnt/c/`

## 💡 **每日实用命令**

```bash

# 4. 后台运行并输出日志
nohup python train.py > train.log 2>&1 &

# 5. 监控GPU使用

```
