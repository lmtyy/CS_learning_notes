# Micro 编辑器基础用法指南

Micro 是一款现代化的终端文本编辑器，设计理念是**简单易用**，同时保留强大的功能。它像 Nano 一样容易上手，但拥有更多现代编辑器的特性。

## 📦 安装 Micro

### Ubuntu/Debian
```bash
# 方法1：使用官方脚本（推荐）
curl https://getmic.ro | bash
sudo mv micro /usr/local/bin/

# 方法2：使用包管理器
sudo apt update
sudo apt install micro
```

### CentOS/RHEL/Fedora
```bash
sudo dnf install micro
# 或
sudo yum install micro
```

## 🚀 基础使用

### 启动和退出
```bash
micro                    # 启动空编辑器
micro 文件名            # 编辑指定文件
micro file1.txt file2.c # 同时打开多个文件
```

**退出编辑器**：按 `Ctrl+Q` 或输入 `quit` 后回车

| **Ctrl+Tab** | 切换标签页 | |

## 📝 编辑模式特性

### 1. **鼠标支持**
- 直接点击移动光标
- 拖动选择文本
- 右键显示上下文菜单
- **注意**：需要终端支持鼠标

### 2. **多光标编辑**（很实用的功能）
```bash
# 创建多光标
Ctrl+鼠标点击      # 在点击位置添加光标
Ctrl+Shift+上下键  # 向上/下添加光标
Alt+鼠标拖动       # 块选择模式

# 操作多光标
所有光标会同步你的输入
按 Esc 退出多光标模式
```

### 3. **语法高亮**
自动识别文件类型，支持：
- Python、JavaScript、Java、C/C++、Go、Rust
- HTML、CSS、Markdown
- JSON、YAML、TOML
- Shell脚本等

## 🔧 常用命令模式操作

按 `Ctrl+E` 进入命令模式，可以输入：

### 文件管理
```bash
# 保存相关
save           # 保存当前文件
save 新文件名  # 另存为
saveall        # 保存所有打开的文件

# 退出相关
quit           # 退出（如有未保存会提示）
quit!          # 强制退出，不保存
```

### 编辑器设置
```bash
# 显示相关
set syntax on          # 开启语法高亮
set syntax off         # 关闭语法高亮
set colorscheme 主题名 # 切换配色方案
set tabstospaces true  # Tab转空格
set tabsize 4          # 设置Tab大小

# 查看设置
show settings          # 显示所有设置
```

### 常用命令
```bash
help            # 打开帮助
help 快捷键     # 查看特定帮助
version         # 显示版本信息
```

## 🎨 配置 Micro

### 1. **创建配置文件**
```bash
# Micro 会自动创建配置目录
~/.config/micro/
    ├── settings.json    # 主配置文件
    ├── bindings.json    # 快捷键绑定
    └── init.lua         # Lua初始化脚本（高级）
```

### 2. **常用配置示例**（编辑 `~/.config/micro/settings.json`）：
```json
{
    "autosave": 1,           // 每1秒自动保存
    "tabstospaces": true,    // Tab转为空格
    "tabsize": 4,            // Tab大小为4空格
    "colorscheme": "simple", // 配色方案
    "cursorline": true,      // 高亮当前行
    "scrollbar": true,       // 显示滚动条
    "softwrap": true,        // 自动换行
    "ruler": true,           // 显示标尺（行列号）
    "savecursor": true,      // 记住光标位置
    "saveundo": true         // 保存撤销历史
}
```

### 3. **常用配色方案**：
- `simple` - 简洁黑白
- `solarized` - Solarized主题
- `monokai` - Monokai主题
- `gruvbox` - Gruvbox主题
- `atom-dark` - Atom暗色主题

切换命令：`set colorscheme monokai`

## 🔍 实用技巧

### 1. **文件浏览器**
```bash
# 打开文件浏览器
Ctrl+O  # 然后按 Tab 切换到文件列表

# 在文件浏览器中：
Enter    # 打开文件/进入目录
..       # 返回上级目录
/        # 输入路径跳转
```

### 2. **分屏编辑**
```bash
# 垂直分屏
Ctrl+E 然后输入：vsplit 文件名

# 水平分屏
Ctrl+E 然后输入：hsplit 文件名

# 切换分屏
Ctrl+W 然后方向键
```

### 3. **快速导航**
```bash
# 跳转到行
Ctrl+G  # 输入行号跳转

# 查找
Ctrl+F  # 输入查找内容
n       # 查找下一个
N       # 查找上一个
```

### 4. **插件系统**
```bash
# 安装插件管理器
micro -plugin install manager

# 安装插件示例
Ctrl+E 然后输入：plugin install go         # Go语言支持
plugin install python                     # Python增强
plugin install filemanager               # 文件管理器
plugin install comment                   # 注释插件
```

## 🆚 与 Nano 对比

| 特性 | Micro | Nano |
|------|-------|------|
| **学习曲线** | 平缓 | 平缓 |
| **鼠标支持** | ✅ | ❌ |
| **多光标** | ✅ | ❌ |
| **现代UI** | ✅ | ❌ |
| **插件系统** | ✅ | ❌ |
| **语法高亮** | 优秀 | 基础 |
| **配置方式** | JSON文件 | 命令行参数 |

**总结**：Micro 是 Nano 的现代化替代品，如果你熟悉图形编辑器的操作，Micro 会更加顺手。它的快捷键设计符合现代习惯，不需要像 Vim 那样记忆模式切换。
