## 🚀 **启动 JupyterLab**

首先在你的机器学习环境中启动：

```bash
conda activate ml_learning
jupyter lab
```

这会自动在浏览器中打开 JupyterLab 界面（通常是 `http://localhost:8888/lab`）。

## 🎯 **界面概览**

JupyterLab 界面主要分为几个区域：

```
+-------------------------------------------------+
| 顶部菜单栏 [File] [Edit] [View] [Run] ...        |
+------------+------------------------------------+
|            |                                    |
| 左侧边栏   |          主工作区                   |
|            |      (标签页形式)                  |
| • 文件浏览器|   [Notebook1] [Terminal] [Text文件] |
| • 运行终端  |                                    |
| • 命令面板  |                                    |
|            |                                    |
+------------+------------------------------------+
```

## 📝 **创建你的第一个 Notebook**

### 步骤1：创建新笔记本
1. 点击左侧边栏的 **"+"** 按钮
2. 在 Launcher 页面点击 **"Python 3"** 下的 Notebook 图标
3. 或者通过菜单：`File` → `New` → `Notebook`

### 步骤2：重命名笔记本
- 右键点击标签页 → `Rename Notebook`
- 或者文件浏览器中右键文件 → `Rename`
- 建议命名：`my_first_machine_learning.ipynb`

## 🎮 **基本操作指南**

### 1. **单元格类型**
每个单元格可以是不同类型：

- **Code**：写 Python 代码
- **Markdown**：写文档、标题、说明文字
- **Raw**：原始文本

通过工具栏下拉菜单切换类型。

### 2. **运行代码**
```python
# 在代码单元格中输入：
print("Hello, Machine Learning!")
import numpy as np
np.array([1, 2, 3])
```

运行方式：
- 点击工具栏的 **"Run"** 按钮
- 快捷键：`Shift + Enter`（运行并跳到下一单元格）
- `Ctrl + Enter`（运行当前单元格）

### 3. **Markdown 文档**
创建一个 Markdown 单元格：
```
# 标题
## 二级标题

这是普通文本

**粗体** *斜体*

- 列表项1
- 列表项2

`行内代码`

数学公式：$E = mc^2$
```

## 🛠️ **实用功能**

### 1. **多任务工作**
你可以同时打开：
- **多个 Notebook**（标签页形式）
- **终端**：`File` → `New` → `Terminal`
- **文本编辑器**：`File` → `New` → `Text File`
- **Markdown 编辑器**

### 2. **文件管理**
- 在左侧文件浏览器中上传、下载、创建文件夹
- 拖拽文件进行整理

### 3. **快捷键**
重要快捷键：
- `A`：在上面插入单元格
- `B`：在下面插入单元格
- `D, D`：删除单元格（按两次D）
- `M`：转换为 Markdown
- `Y`：转换为 Code
- `Ctrl + S`：保存

查看所有快捷键：`Help` → `Show Keyboard Shortcuts`

## 🎯 **机器学习实战示例**

创建一个完整的机器学习工作流：

### 单元格1：Markdown
```markdown
# 我的第一个机器学习项目
## 数据加载和探索
```

### 单元格2：代码
```python
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt

print("环境检查完成！")
```

### 单元格3：代码
```python
# 生成示例数据
from sklearn.datasets import make_classification

X, y = make_classification(n_samples=1000, n_features=20, n_classes=2, random_state=42)
print(f"数据形状: X{X.shape}, y{y.shape}")
```

### 单元格4：代码
```python
# 数据可视化
plt.figure(figsize=(10, 6))
plt.scatter(X[:, 0], X[:, 1], c=y, cmap='viridis')
plt.title("数据分布")
plt.colorbar()
plt.show()
```

## 🔧 **高级技巧**

### 1. **魔法命令**
在单元格中使用特殊命令：
```python
%timeit np.sum(np.random.rand(1000, 1000))  # 测量运行时间

%matplotlib inline  # 内嵌显示图表

!pip list  # 运行系统命令（查看已安装包）
```

### 2. **自动补全**
- 输入部分代码后按 `Tab` 键
- 查看函数文档：在函数名后按 `Shift + Tab`

### 3. **内核管理**
- 重启内核：`Kernel` → `Restart Kernel`
- 中断运行：`Kernel` → `Interrupt Kernel`
