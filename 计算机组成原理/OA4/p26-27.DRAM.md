### 详细讲解

#### 一、静态RAM与动态RAM

课件首先回顾并对比了两种主要的随机存取存储器。

**1. 静态RAM**

*   **核心元件**：如你之前所学，是**六晶体管触发器**。可以把它想象成一个非常稳固的“跷跷板”，一旦拨到一边，它就会稳定地保持在那一边的状态（代表0或1），除非你用力把它拨到另一边。
*   **特点**：
    *   **不需要刷新**：只要保持通电，信息就能一直保存，不需要额外操作。
    *   **速度快**：因为状态稳定，读写访问非常迅速。
    *   **工作稳定可靠**：电路简单，控制逻辑不复杂。

**2. 动态RAM - 本课重点**

*   **核心元件**：一个**MOS管**和一个**电容**。这是关键！
    *   **电容**：负责存储电荷。**有电荷代表1，无电荷代表0**。
    *   **MOS管**：相当于一个开关，控制外部电路能否访问这个电容进行充放电。
*   **工作原理**：
    1.  **写入**：打开MOS管开关，向电容充电（写1）或放电（写0）。
    2.  **保持**：关闭开关，电荷理论上应该保存在电容里。
    3.  **读取**：打开开关，检测电容上的电压（电荷量）来判断是0还是1。
*   **核心问题与“动态”的由来**：
    *   电容就像一个有微小裂缝的水桶。即使关闭了开关（MOS管），电荷也会通过MOS管的**漏电阻** 慢慢泄漏掉。
    *   这会导致：存储的“1”会慢慢变成“0”，信息就丢失了！
    *   **解决方案**：**刷新**。必须在电荷泄漏到无法识别之前，定期地、**周期性地** 把数据读出来，然后再重新写回去（相当于给电容“充电”）。
    *   正是因为需要这种周期性的、动态的刷新操作来维持数据，所以它被称为 **Dynamic RAM**。

#### 二、为什么提出动态存储单元

这一部分解释了从SRAM发展到DRAM的技术驱动力。

**SRAM的优点（也是DRAM的缺点）**：
*   **工作稳定，无需刷新**：正如第一页所说，这是SRAM最大的优点。

**SRAM的缺点（也是DRAM的优点）**：
*   **功耗大**：一个SRAM单元需要6个晶体管。无论是否被读写，只要通电，这6个管子中总有一些处于导通状态，会产生持续的静态电流，导致功耗较大。
*   **集成度低**：6个晶体管占用的芯片面积远比“1个晶体管+1个电容”要大。

**DRAM的优势**：
*   **集成度高**：DRAM的单细胞结构非常简单（1T1C），在同样面积的硅片上可以制造出更多存储单元。这就是为什么我们能有**GB级别**的内存条，而同样技术的SRAM容量会小得多。
*   **功耗低**：主要功耗来自刷新和读写，静态功耗（保持状态时）远低于SRAM。
*   **成本低**：结构简单，集成度高，自然成本就更低。

所以，课件第二页的结论非常精辟：**DRAM单元电路恰好克服了（SRAM功耗大、集成度低的）缺点。DRAM的出现是半导体存储技术的一大进步。**

### 核心总结与类比

为了让你记得更牢，我们可以做一个简单的类比：

| 项目 | **SRAM** | **DRAM** |
| :--- | :--- | :--- |
| **结构** | 六管触发器（复杂） | 一管一电容（简单） |
| **原理** | 稳态电路，像**跷跷板** | 电容充电，像**漏水的桶** |
| **刷新** | **不需要** | **需要定期刷新** |
| **速度** | **快** | 较慢（需要刷新，访问延迟） |
| **集成度** | 低（占地方） | **高**（更密集） |
| **功耗** | 较大 | 较小 |
| **成本** | 高 | **低** |
| **应用** | **CPU高速缓存** | **主内存** |

**结论**：正是因为DRAM在**成本、集成度和功耗**上的巨大优势，尽管它速度慢且需要复杂的刷新电路，它依然成为了制造计算机**主内存**的不二之选。而SRAM则凭借其**极致的速度**，被用在要求极高的**CPU缓存**中。
