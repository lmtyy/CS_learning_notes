非常好！这几张幻灯片介绍了快速以太网的另一个重要成员——**100BASE-FX**。这是为需要**更长距离、更强抗干扰能力**的环境设计的解决方案，它让我们看到光纤是如何开始融入以太网体系的。

---

### **100BASE-FX 概览**

*   **命名解读**：
    *   **100**： 100 Mbps 速率
    *   **BASE**： 基带传输
    *   **FX**： 使用 **光纤** 介质
*   **核心特征**：
    *   **介质**：使用 **两束多模光纤**，一束用于发送（TX），一束用于接收（RX）。这是标准的全双工物理配置。
    *   **协议**：仍然使用 **CSMA/CD** 访问方法（在需要兼容半双工模式时）。
    *   **主要优势**：相比双绞线，光纤提供了**远得多的传输距离**和**完全的电磁免疫性**。

---

### **关键参数：为什么是412米和228米？**

这是100BASE-FX最需要理解的技术点，它直接源于我们熟悉的 **CSMA/CD约束公式**。

**公式回顾**：
`Lmin = (2S / 0.7C + 2t_PHY + 2N*t) x R`
其中：
*   `Lmin` = 最小帧长（512比特）
*   `S` = 网络最大端到端距离（最远两台设备间的光纤长度）
*   `t_PHY` = 物理层设备（如收发器）延迟
*   `N` = 中继器数量
*   `t` = 每个中继器的延迟
*   `R` = 网络速率（100 Mbps）

对于100BASE-FX，主要讨论的是**不使用中继器**（即 `N=0`）的连接场景，也就是 **DTE（数据终端设备，如交换机或网卡）之间直接通过光纤相连**。

---

#### **1. 半双工模式下的最大长度：412米**

*   **模式**：**半双工**。在这种模式下，设备不能同时收发，且必须遵守CSMA/CD协议来检测冲突。这意味着信号需要在最远距离上跑一个来回（`2S`）。
*   **计算目标**：在保持**最小帧长（64字节）**和**争用期（51.2μs）**不变的前提下，计算光纤能允许的最大距离 `S`。
*   **结果**：根据公式计算，并考虑光纤中光速传播系数（约为真空光速的0.7倍）和收发器延迟，得出在半双工模式下，**两个DTE设备之间直接通过光纤连接的最大距离为 412米**。
*   **应用场景**：这个模式主要用于早期为了兼容传统共享式网络架构，或者连接老式的光纤集线器。

#### **2. 全双工模式下的有效距离**

*   **模式**：**全双工**。这是光纤连接**最主要、最推荐**的工作模式。
*   **关键区别**：在全双工模式下，设备可以同时收发，**CSMA/CD协议被禁用**。因为没有冲突需要检测，所以**不受51.2μs争用期的距离限制**！
*   **结果**：距离限制仅由**光纤本身的信号衰减特性**和**收发器的功率**决定。根据100BASE-FX标准，使用标准的多模光纤，**全双工点对点连接的最大距离可达 2公里**。
*   **图中“228米”的解读**：这个数字可能是某种特定、更严苛的配置下的计算结果（例如，连接了特定的中继器 `N>0`），或者是一个保守的工程参考值。但它**不是全双工模式的标准最大距离**。全双工下的2公里才是其优势所在。

### **与100BASE-TX的对比**

| 特性 | **100BASE-TX** | **100BASE-FX** |
| :--- | :--- | :--- |
| **介质** | CAT 5 双绞线 | 多模光纤 |
| **最大距离（半双工）** | 100米（集线器到设备） | **412米**（DTE到DTE） |
| **最大距离（全双工）** | 100米（仍受限于线缆衰减） | **2公里** |
| **抗干扰性** | 较弱，受电磁干扰 | **极强，完全免疫电磁干扰** |
| **安全性** | 信号可被窃听 | **物理窃听非常困难，安全性高** |
| **成本** | 低 | 较高（光纤、光模块均更贵） |
| **主要用途** | 桌面连接、楼层布线 | 建筑物间骨干连接、车间、电磁环境恶劣的区域 |

---

### **总结**

**100BASE-FX是以太网向高性能、远距离、严苛环境应用迈出的重要一步。**

1.  **它保留了快速以太网的核心**（100M速度，MAC帧格式）。
2.  **它通过光纤介质**，突破了双绞线的百米距离瓶颈，为构建园区网骨干提供了理想方案。
3.  **它凸显了工作模式的重要性**：在半双工下，它仍受CSMA/CD的约束（412米）；一旦切换到**全双工**，就解放了距离限制，性能达到最佳（2公里）。这加速了全双工和交换机在骨干网络中的普及。

因此，100BASE-FX常见于连接不同大楼的交换机、数据中心机架间的互连，或者在工厂、医院等存在强电磁干扰的环境中。它和100BASE-TX一起，构成了快速以太网时代“铜缆到桌面，光纤做骨干”的典型网络架构。
