### Advantage（幻灯片列出的优点）
- Security  
  - Stops networks from forwarding sensitive data（阻止网络将敏感数据无差别转发到其它段）。  
- Bandwidth  
  - Reduce traffic by segmentation（通过分段减少不必要的流量以提高带宽利用）。  
- Reliability  
  - If one segment goes down, it does not take down the complete LAN（段故障不会使整个 LAN 瘫痪）。  
- Translation  
  - Translate different Data Link protocols such as Token Ring to Ethernet（能在不同链路层协议间做简单的桥接/转换，如 Token Ring ↔ Ethernet，幻灯片只列出此用途）。
