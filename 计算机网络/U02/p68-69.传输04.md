### **第一页：四种转换场景与核心设备**

这张图展示了数据通信的四种基本模式，以及每种模式下需要的关键设备。

#### **1. 模拟数据 -> 模拟信号**
*   **路径：** `Voice` -> `Telephone` -> `Analog Transmission`
*   **过程：** 人的声音（**模拟数据**）直接被电话听筒转换成电信号（**模拟信号**），然后通过模拟线路（如传统电话线）传输。
*   **核心设备：** **电话**。它进行的是简单的电-声转换，没有数字化的过程。
*   **特点：** 最传统的方式，抗干扰能力差。

#### **2. 模拟数据 -> 数字信号**
*   **路径：** `Voice` -> `CODEC` -> `Digital Transmission`
*   **过程：** 人的声音（**模拟数据**）经过 **CODEC** 设备，被数字化（采样、量化、编码）成**数字信号**，然后通过数字网络传输。
*   **核心设备：** **CODEC**。这个名字来源于 **Coder（编码器）** 和 **DECoder（解码器）** 的合成。它负责实现**模数转换**和**数模转换**。
*   **特点：** 现代语音通信（如IP电话、移动电话）的基础，传输质量高。

#### **3. 数字数据 -> 模拟信号**
*   **路径：** `Digital` -> `Modem` -> `Analog Transmission`
*   **过程：** 计算机产生的**数字数据** 通过 **Modem** 设备，被调制成**模拟信号**，以便在模拟信道（如电话线、有线电视线）上传输。
*   **核心设备：** **Modem**。这个名字来源于 **Modulator（调制器）** 和 **DEModulator（解调器）** 的合成。
*   **特点：** 利用现有模拟设施进行数字通信的关键，如早期的拨号上网。

#### **4. 数字数据 -> 数字信号**
*   **路径：** `Digital` -> `Digital Transmitter` -> `Digital Transmission`
*   **过程：** 计算机产生的**数字数据** 通过**数字发送器**（如网卡），进行**编码**（如曼彻斯特编码、4B/5B编码）后，变成**数字信号**（方波）在数字信道（如网线、光纤）中传输。
*   **核心设备：** **数字发送器/接收器**（如网卡）。它进行的是**数字到数字**的编码。
*   **特点：** 抗干扰能力强，是现代局域网和设备间通信的主流方式。

### **第二页：编码与调制的技术原理**

这张图从技术细节上区分了 **编码** 和 **调制** 这两个核心过程。

#### **左侧：编码**
*   **过程：** `g(t) [analog]` -> **Encoder** -> `x(t) [digital]`
*   **功能：** 将**模拟信号** `g(t)` 转换为**数字信号** `x(t)`。这就是我们刚讲的 **PCM（脉冲编码调制）** 的过程，即**模数转换**。
*   **核心：** **采样、量化、编码**。
*   **逆过程：** **解码**，将数字信号 `x(t)` 恢复为模拟信号 `g(t)`。

#### **右侧：调制**
*   **过程：** `m(t) [digital]` -> **Modulator** -> `s(t) [analog]`
*   **功能：** 将**数字信号** `m(t)`（基带信号）“装载”到一个高频的**载波** `f_c` 上，生成一个适合在模拟信道传输的**模拟信号** `s(t)`。
*   **核心：** 通过改变载波的**幅度、频率或相位**来代表数字比特0和1（即ASK, FSK, PSK）。
*   **频谱变化：** 下方的频谱图 `s(f)` 显示，调制将基带信号 `m(t)` 的频谱从低频搬移到了载波频率 `f_c` 的周围。这使得多路信号可以通过**频分复用**在同一信道上传输。
*   **逆过程：** **解调**，从已调信号 `s(t)` 中还原出数字基带信号 `m(t)`。

### **全局总结**

这两页课件结合在一起，构成了一个完整的知识体系：

1.  **明确了四种通信场景**及其**核心转换设备**：
    *   **CODEC**：处理 **模拟 <-> 数字** 转换（对应编码/解码）。
    *   **Modem**：处理 **数字 <-> 模拟** 转换（对应调制/解调）。

2.  **从技术原理上区分了“编码”和“调制”**：
    *   **编码** 的核心是**模数转换**，将连续信号变为离散比特。
    *   **调制** 的核心是**频谱搬移**，将数字比特变为模拟波形以适应信道。

**一个现代通信的典型路径（如手机上网）就是这些技术的组合：**
你的语音（模拟数据）-> **CODEC**（编码为数字数据）-> 手机基带芯片（对数字数据进行**调制**）-> 天线（发射模拟射频信号）-> ... （传输）... -> 网络服务器。
