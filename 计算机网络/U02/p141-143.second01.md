### 第一页：EIA/TIA-568 标准简介

这一页介绍了该标准的制定者和核心内容。

*   **制定者**：
    *   **EIA**：美国电子工业协会
    *   **TIA**：美国电信工业协会
    *   这是一个由行业专家共同制定的、确保兼容性的权威标准。

*   **核心内容**：
    *   对**双绞线**的规范说明。
    *   将双绞线电缆根据性能分割成**若干类别**。这正是我们之前学过的 **Cat 5, Cat 5e, Cat 6** 等类别的来源。

---

### 第二、三页：核心知识——568A与568B线序

这两页是重中之重，解释了如何排列双绞线中的8根彩色细线。

#### 1. 为什么需要标准线序？

网卡和交换机（HUB）的接口针脚定义是固定的：
*   **1、2 针脚** 用于 **发送数据**。
*   **3、6 针脚** 用于 **接收数据**。

为了建立通信，一方的“发送”必须连接到另一方的“接收”。标准线序就是为了确保这一点。

#### 2. 两种标准：T568A 与 T568B

课件中的图表和色块清晰地展示了两种国际标准的线序排列：

**线对与颜色标识：**
*   **线对1**：白蓝 / 蓝
*   **线对2**：白橙 / 橙
*   **线对3**：白绿 / 绿
*   **线对4**：白棕 / 棕

**线序对比（从引脚1到引脚8）：**

| 引脚 | 信号 | **T568B 线序** | **T568A 线序** |
| :--- | :--- | :--- | :--- |
| 1 | **TX+** | **白橙** | **白绿** |
| 2 | **TX-** | **橙** | **绿** |
| 3 | **RX+** | **白绿** | **白橙** |
| 4 | - | 蓝 | 蓝 |
| 5 | - | 白蓝 | 白蓝 |
| 6 | **RX-** | **绿** | **橙** |
| 7 | - | 白棕 | 白棕 |
| 8 | - | 棕 | 棕 |

**关键点**：
*   两种标准的主要区别在于**橙色线对**和**绿色线对**的位置交换了。
*   **T568B** 在中国和北美地区更为常用。
*   **在一个工程中，必须统一使用一种标准，不能混用！**

### 线序类型的实际应用

根据线序两端是否相同，网线分为两种：

#### 1. 直通线
*   **线序**：两端都采用 **T568A** 或两端都采用 **T568B**。
*   **作用**：连接**不同类**设备。
*   **典型应用**：
    *   **计算机** <--> **交换机/路由器**
    *   **路由器** <--> **交换机**

#### 2. 交叉线
*   **线序**：一端用 **T568A**，另一端用 **T568B**。
*   **作用**：连接**同类**设备。
*   **原理**：通过交叉线序，将一端的发送线对（1,2）连接到另一端的接收线对（3,6），实现直接通信。
*   **典型应用**：
    *   **计算机** <--> **计算机**
    *   **交换机** <--> **交换机**

**现代设备的自动化**：现在很多网络设备支持 **Auto-MDI/MDIX** 功能，可以自动检测线序并调整，所以用直通线也可以连接同类设备。但理解其原理仍然至关重要。

### 总结

**EIA/TIA-568** 标准通过定义 **568A** 和 **568B** 这两种线序，为全球的双绞线以太网布线提供了统一的、可靠的物理连接方案。它确保了：
1.  **正确的信号匹配**：发送端对接收端。
2.  **最佳的串扰抑制**：通过将一对双绞线用于一个差分信号，并将不同线对保持不同的绞合度，来最大限度地减少干扰。

记住这两种线序，尤其是你所在地区最常用的那种（很可能是 **T568B**），是进行网络布线和故障排查的基本功。
