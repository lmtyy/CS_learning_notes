好的，这部分课件是关于计算机网络的两种最基本、最重要的分类方式：**按交换技术** 和 **按地理范围**。

这两种分类方式从不同维度描述了网络的特征，是理解网络如何工作和它们规模大小的基础。

---

### 分类一：按交换技术

交换技术决定了数据在网络中是如何从源头传送到目的地的。主要有三种：

#### 1. 电路交换

*   **工作方式**：在通信开始之前，必须在发送端和接收端之间**建立一条专用的物理通信路径**。这条路径在整个通信过程中都将被完全占用，直到通信结束才释放。
*   **经典类比**：**传统电话系统**。当你打电话时，运营商会在你和对方之间建立一条物理线路，这条线路在通话期间专属于你们，即使你们都不说话（没有数据传输），别人也无法使用。
*   **特点**：
    *   **优点**：一旦建立，数据传输的延迟非常小且稳定。
    *   **缺点**：线路利用率低，因为专用线路在静默期也被占用。

#### 2. 报文交换

*   **工作方式**：整个要发送的**完整数据块（称为“报文”）** 被一次性发送到网络中。每个中间节点（如路由器）接收并存储整个报文，然后根据目标地址将其转发到下一个节点。这被称为 **“存储-转发”** 网络。
*   **经典类比**：**邮政系统**。你写一封信（报文），投入邮箱。邮局（网络节点）会收集这封信，然后根据地址决定如何路由，最终送到收件人手中。
*   **特点**：
    *   **优点**：不需要建立专用线路，线路利用率比电路交换高。
    *   **缺点**：每个中间节点都需要存储整个报文，对节点缓存要求高；大报文会长时间占用链路，导致延迟增加。

#### 3. 分组交换

*   **工作方式**：这是**现代互联网的核心技术**。它改进了报文交换。
    *   首先，将需要传输的完整数据**分割成一个个小的、长度固定的数据块**，这些数据块称为 **“分组”或“包”**。
    *   然后，每个分组独立地在网络中传输。它们可能经过不同的路径，最终在目的地被重新组装成原始数据。
*   **经典类比**：**一辆大卡车（大数据）的货物被分装到多辆小货车上（分组）。** 这些小货车可以走不同的高速公路（网络路径）以避开拥堵，最终在目的地仓库被重新组装。
*   **特点**：
    *   **优点**：
        *   **高效**：链路可以被多个不同通信的分组共享，利用率极高。
        *   **健壮**：如果某条路径中断，分组可以绕行其他路径。
    *   **缺点**：由于分组可能走不同路径，会产生**乱序**和**延迟抖动** 的问题，需要更复杂的协议（如TCP）来处理。

**小结**：今天的互联网主要基于**分组交换**。

### 分类二：按地理范围

这种分类方式直观地描述了网络的覆盖规模。

#### 1. LAN - 局域网
*   **范围**：较小，通常是一个**建筑物、校园或几公里内的区域**。
*   **所有权**：通常为**私人或单位所有、管理和维护**。
*   **特点**：传输速率高、延迟低、误码率低。
*   **例子**：你家里的Wi-Fi网络，办公室或学校机房的网络。

#### 2. MAN - 城域网
*   **范围**：覆盖**整个城市**。
*   **特点**：可以看作是多个LAN的互联，通常由电信运营商或市政部门建设。
*   **例子**：城市范围内的有线电视网络、某些城市级的政府专网。

#### 3. WAN - 广域网
*   **范围**：覆盖**很大的地理区域**，如一个国家、一个洲乃至全球。
*   **特点**：通常使用运营商提供的公共通信线路，速度可能低于LAN，延迟较高。
*   **例子**：**互联网本身就是一个最大的WAN**。一个跨国公司在不同国家分公司的内部网络也是一个WAN。

### 总结与关联

| 分类方式 | 类型 | 核心思想 | 典型应用 |
| :--- | :--- | :--- | :--- |
| **按交换技术** | 电路交换 | **先建立专用通路**，通信期间独占 | 传统电话 |
| | 报文交换 | **存储-转发整个报文** | 电报系统（已过时） |
| | **分组交换** | **将数据分块，独立路由** | **现代互联网 (TCP/IP)** |
| **按地理范围** | **LAN (局域网)** | 范围小，私人拥有，高速 | 家庭、办公室网络 |
| | MAN (城域网) | 覆盖一个城市 | 城市宽带 |
| | **WAN (广域网)** | 范围大，跨越地理区域 | **互联网**，跨国企业网 |

这两种分类是正交的。例如，一个**LAN（局域网）** 内部可能使用电路交换技术（如早期的电话交换机），但今天绝大多数LAN和整个**WAN（互联网）** 都使用的是**分组交换**技术。
