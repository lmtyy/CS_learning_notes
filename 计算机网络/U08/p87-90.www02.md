这四张课件深入探讨了**万维网 (WWW)** 的核心技术——**超文本 (Hypertext)** 与 **超媒体 (Hypermedia)**，解释了网页是如何从单纯的文字演变为如今丰富多彩的交互形态的。

以下是详细解读：

---

### 1. 超文本 (Hypertext) 的概念

课件 **image_52f027.jpg** 定义了 Web 文档的基本结构：

* **非顺序性 (Non-linear)**：与传统书籍必须从第一页读到最后一页不同，超文本允许你跳跃式地阅读不同的段落或章节。
* **指针连接 (Links)**：文档内部包含“指针”，点击它们可以跳转到同一文档的其他位置，甚至是**网络上另一台主机**上的文档。
* **分布式广泛**：资源不再局限于本地，而是遍布全球网络。

### 2. 阅读格式的对比

课件 **image_52f043.png** 用图示形象地对比了两种阅读逻辑：

* **顺序阅读格式 (左图)**：线性流动，从第一章到第二章，路径单一固定。
* **非顺序阅读格式 (右图)**：这就是我们现在的网页。你可以从目录直接跳到第二章，或者从第二章的某个关键词跳到完全无关的外部参考文档。

### 3. 超媒体 (Hypermedia) 的进化

课件 **image_52f048.jpg** 展示了信息的表现形式是如何升级的：

* **媒体 (Media)**：信息的最基本表现形式。
* **多媒体 (Multimedia)**：融合了**文本、声音、图像、视频和动画**。
* **超媒体 (HyperMedia)**：这是万维网的终极形式。它是 **“超文本 + 多媒体”** 的结合体。
* *例子*：你点击一张“建筑图片”，不仅能看到文字介绍，还能听到它的语音讲解。



### 4. 万维网的分布式服务架构

课件 **image_52f066.jpg** 展示了全球网页是如何织成“网”的：

* **相互链接**：站点 A 的文档链接到站点 B，站点 B 又链接到 C、D 或 E。
* **网状拓扑**：没有中心节点，每一个 Web 站点（如站点 A 到 E）都是这个分布式服务系统的一部分。

---

### 💡 核心知识点总结

| 概念 | 重点特征 |
| --- | --- |
| **超文本** | 非顺序、跳跃式阅读，通过链接(Pointer)连接。 |
| **超媒体** | 赋予了多媒体（音视频）“可跳转”的链接属性。 |
| **WWW 的本质** | 利用超文本/超媒体技术构建的全球分布式信息系统。 |
