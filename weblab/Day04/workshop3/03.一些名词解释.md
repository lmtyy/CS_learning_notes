# Vite、Node.js、nodemon 是什么？

直接讲核心，不绕弯子。

---

## 一、Node.js（基础平台）

**一句话：让 JavaScript 能在电脑上直接运行，而不是只在浏览器里。**

### 有什么用？
1. **运行服务器**（后端）
   ```javascript
   // server.js
   const http = require('http');
   const server = http.createServer((req, res) => {
     res.end('Hello World');
   });
   server.listen(3000);
   ```
   ```bash
   node server.js  # 启动一个服务器
   ```

2. **运行各种工具**（如 Vite、nodemon 本身也是 Node.js 程序）

### 简单理解：
- 浏览器里的 JavaScript：操作网页
- Node.js 里的 JavaScript：操作电脑文件、网络、数据库

---

## 二、Vite（前端开发工具）

**一句话：帮你快速搭建和开发前端项目。**

### 解决什么问题？
以前开发前端很麻烦：
1. 改一点代码 → 等几十秒才能看到效果
2. 项目大了 → 启动要几分钟
3. 配置复杂 → webpack 配置让人头疼

Vite 让你：
```bash
# 1. 创建项目（几秒钟）
npm create vite@latest

# 2. 安装依赖
npm install

# 3. 启动开发服务器（瞬间）
npm run dev
# 打开 http://localhost:5173
```

### Vite 的两个核心功能：

#### 1. 开发服务器（快）
```bash
npm run dev
```
- 秒启动
- 改代码 → 浏览器立即更新（不用刷新页面）
- 支持 Vue、React、原生HTML等各种项目

#### 2. 打包工具
```bash
npm run build
```
- 把源代码压缩、优化
- 生成 `dist` 文件夹
- 里面的文件可以直接放到服务器上运行

---

## 三、nodemon（开发辅助工具）

**一句话：自动重启 Node.js 程序。**

### 解决什么问题？
开发后端时：
```bash
# 传统方式
node server.js
# 改了代码后要：
# 1. 按 Ctrl+C 停止
# 2. 重新运行 node server.js
```

用 nodemon：
```bash
nodemon server.js
# 改了代码 → 自动重启
# 不用手动停止再启动
```

### 安装和使用：
```bash
# 全局安装
npm install -g nodemon

# 或项目内安装
npm install nodemon --save-dev

# 使用
nodemon server.js
```

### 在 package.json 里配置：
```json
{
  "scripts": {
    "start": "nodemon server.js",
    "dev": "nodemon server.js"
  }
}
```
然后：
```bash
npm start  # 自动用 nodemon 启动
```

## 四、它们的关系

### 开发一个完整项目时：

```
你的电脑
├── Node.js（环境，已安装）
├── Vite（前端开发）
│   ├─ npm run dev（启动前端：端口5173）
│   └─ npm run build（打包前端）
│
├── nodemon（后端开发辅助）
│   └─ npm start（启动后端：端口3000）
│
└── 浏览器
    ├─ 访问 http://localhost:5173（前端）
    └─ 前端调用 http://localhost:3000/api（后端）
```

### 实际工作流：
1. **装 Node.js**（一次性）
2. **用 Vite 创建前端项目**
3. **用 nodemon 开发后端**
4. 前端和后端同时运行，互相调用

---

## 五、简单对比

| 工具 | 是什么 | 什么时候用 | 简单命令 |
|------|-------|-----------|----------|
| **Node.js** | JavaScript 运行环境 | 运行任何 JS 程序（服务器、工具） | `node 文件名.js` |
| **Vite** | 前端开发工具 | 开发 Vue/React/原生前端项目 | `npm run dev` |
| **nodemon** | 自动重启工具 | 开发 Node.js 后端时 | `nodemon server.js` |

## 六、快速开始示例

### 1. 创建一个前后端项目
```bash
# 创建前端（用 Vite）
npm create vite@latest my-app
cd my-app
npm install

# 创建后端文件
echo "const express = require('express');
const app = express();
app.get('/api', (req, res) => {
  res.json({ message: '来自后端的数据' });
});
app.listen(3000, () => console.log('后端运行在3000端口'));" > server.js

# 安装后端依赖
npm install express

# 安装 nodemon（开发用）
npm install nodemon --save-dev
```

### 2. 修改 package.json
```json
{
  "scripts": {
    "dev": "vite",              // 前端
    "build": "vite build",      // 打包前端
    "start": "nodemon server.js" // 后端
  }
}
```

### 3. 运行
```bash
# 终端1：启动后端
npm start
# 输出：后端运行在3000端口

# 终端2：启动前端  
npm run dev
# 输出：http://localhost:5173
```

### 4. 测试
- 浏览器访问 `http://localhost:5173`（前端）
- 前端代码里调用 `http://localhost:3000/api`（后端）

---

## 总结

- **Node.js**：基础，让 JS 能在服务器运行
- **Vite**：前端开发工具，快且简单
- **nodemon**：开发时自动重启后端

**记住**：Vite 管前端，nodemon 管后端开发体验，都运行在 Node.js 上。
