# 🌟 给初学者的请求方法：3种必须掌握的

别担心！作为初学者，你只需要先掌握这3种最常用、最核心的方法：

## 一、📝 **表单提交**（最简单，先学这个）

这是最传统的发送请求方式，适合初学者理解基本概念。

```html
<!-- 1. GET请求表单（搜索、查询） -->
<form action="/search" method="GET">
  <input type="text" name="keyword" placeholder="请输入关键词">
  <button type="submit">搜索</button>
</form>
<!-- 效果：浏览器会跳转到 /search?keyword=输入的内容 -->

<!-- 2. POST请求表单（登录、注册） -->
<form action="/login" method="POST">
  <input type="text" name="username" placeholder="用户名">
  <input type="password" name="password" placeholder="密码">
  <button type="submit">登录</button>
</form>
<!-- 效果：提交到服务器，页面会刷新 -->
```

**特点**：
- ✅ 最简单，写几行HTML就能用
- ✅ 自动处理数据格式
- ❌ 会刷新页面（用户体验不好）
- ❌ 只能发送简单的文本数据

## 二、🔄 **Fetch API**（现代标准，必须学）

这是现在最推荐的方法，功能强大且语法简洁。

### **基本用法（三步走）**
```javascript
// 第1步：发送GET请求获取数据
fetch('https://api.example.com/users')
  .then(response => response.json())  // 第2步：解析为JSON
  .then(data => {                     // 第3步：使用数据
    console.log('获取的用户数据:', data);
  })
  .catch(error => {
    console.error('请求失败:', error);
  });

// 发送POST请求（提交数据）
fetch('https://api.example.com/login', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    username: '张三',
    password: '123456'
  })
})
.then(response => response.json())
.then(data => console.log('登录结果:', data));
```

**记住这个模板**：
```javascript
fetch('URL地址', {
  method: '请求方法',  // GET、POST、PUT、DELETE
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify(你的数据)
})
.then(响应 => 响应.json())
.then(数据 => 处理数据)
.catch(错误 => 处理错误);
```

## 三、✨ **Axios**（最好用，推荐学）

第三方库，比Fetch更简单、功能更全，实际工作中很常用。

### **安装和使用**
```html
<!-- 在HTML中引入 -->
<script src="https://cdn.jsdelivr.net/npm/axios/dist/axios.min.js"></script>
```

```javascript
// 1. GET请求（最简单）
axios.get('/api/users')
  .then(response => {
    console.log(response.data); // 数据在这里！
  });

// 2. POST请求
axios.post('/api/login', {
  username: '张三',
  password: '123456'
})
.then(response => {
  console.log('登录成功:', response.data);
});

// 3. 处理错误（比Fetch方便）
axios.get('/api/users')
  .then(response => console.log(response.data))
  .catch(error => {
    if (error.response) {
      // 服务器返回了错误
      console.log('状态码:', error.response.status);
      console.log('错误信息:', error.response.data);
    }
  });
```

## 📊 对比总结

| 方法 | 优点 | 缺点 | 使用场景 |
|------|------|------|----------|
| **表单提交** | 简单、无需JS | 会刷新页面、功能有限 | 简单联系表单、搜索框 |
| **Fetch API** | 原生支持、功能强大 | 错误处理稍复杂 | 现代项目、需要原生API时 |
| **Axios** | 使用简单、功能全面 | 需要额外引入 | 实际项目开发首选 |

## 🎯 给初学者的学习建议

### **第一步（第1周）**
先掌握**表单提交**，理解：
- GET和POST的区别
- 数据是如何发送到服务器的
- 服务器如何接收数据

### **第二步（第2-3周）**
学习**Fetch API**：
- 使用浏览器的开发者工具（F12 → Network标签）
- 查看请求和响应的具体内容
- 练习发送GET和POST请求

### **第三步（第3-4周）**
学习**Axios**：
- 在实际小项目中应用
- 练习错误处理
- 了解如何发送JSON数据

## 💡 实战小练习

### **练习1：用户登录表单**
```html
<!-- 方法1：传统表单 -->
<form id="loginForm" action="/api/login" method="POST">
  <input type="email" name="email" required>
  <input type="password" name="password" required>
  <button type="submit">登录</button>
</form>

<!-- 方法2：使用Fetch（不刷新页面） -->
<form id="loginForm2">
  <input type="email" id="email" required>
  <input type="password" id="password" required>
  <button type="button" onclick="login()">登录</button>
</form>

<script>
async function login() {
  const email = document.getElementById('email').value;
  const password = document.getElementById('password').value;
  
  try {
    const response = await fetch('/api/login', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ email, password })
    });
    
    const result = await response.json();
    if (result.success) {
      alert('登录成功！');
      // 跳转到首页
      window.location.href = '/dashboard';
    } else {
      alert('登录失败：' + result.message);
    }
  } catch (error) {
    alert('网络错误，请重试');
  }
}
</script>
```
