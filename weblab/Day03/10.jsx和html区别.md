## 核心区别

**JSX是JavaScript的语法扩展，不是HTML**。它最终会被编译成JavaScript函数调用。

## 主要区别点

### 1. 语法规则不同
**HTML**
```html
<div class="container" tabindex="1">
  <label for="name">姓名</label>
</div>
```

**JSX**
```jsx
<div className="container" tabIndex="1">
  <label htmlFor="name">姓名</label>
</div>
```
*区别：JSX使用驼峰命名，避免与JavaScript关键字冲突*

### 2. 嵌入JavaScript表达式
**JSX可以在{}中写JavaScript**
```jsx
function Greeting({ name }) {
  return <h1>你好，{name}！</h1>;
}
```

### 3. 必须闭合所有标签
**JSX要求所有标签必须闭合**
```jsx
<img src="..." alt="..." />  {/* 自闭合标签 */}
<input type="text" />        {/* 自闭合标签 */}
<div>...</div>               {/* 成对标签 */}
```

### 4. 返回单个根元素
**JSX只能返回一个根元素**
```jsx
// ✅ 正确
return (
  <div>
    <h1>标题</h1>
    <p>内容</p>
  </div>
);

// ❌ 错误
return (
  <h1>标题</h1>
  <p>内容</p>
);
```

## 本质区别

- **HTML**：是标记语言，直接由浏览器解析
- **JSX**：是语法糖，会被编译成 `React.createElement()` 调用

```jsx
// JSX写法
const element = <h1 className="title">Hello</h1>;

// 编译后的JavaScript
const element = React.createElement('h1', { className: 'title' }, 'Hello');
```

记住：**JSX是给React用的模板语法，不是HTML**。
