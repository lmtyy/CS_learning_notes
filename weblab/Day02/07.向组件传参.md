## 传参的基本方法

### 方法：使用 "属性"（props）
```jsx
// 定义组件时接收参数
function Welcome(props) {
  return <h1>你好，{props.name}！</h1>;
}

// 使用组件时传参
<Welcome name="小明" />
<Welcome name="小红" />
<Welcome name="李老师" />
```

**显示结果：**
```
你好，小明！
你好，小红！
你好，李老师！
```

## 更简洁的写法：解构赋值

上面的 `props.name`、`props.age` 写起来有点长，可以用更简洁的方式：

```jsx
// 原来的写法
function StudentCard(props) {
  return (
    <div>
      <h3>{props.name}</h3>
      <p>年龄：{props.age}</p>
    </div>
  );
}

// 简洁的写法（推荐！）
function StudentCard({ name, age, grade, score }) {
  return (
    <div>
      <h3>{name}</h3>
      <p>年龄：{age}</p>
      <p>班级：{grade}</p>
      <p>成绩：{score}分</p>
    </div>
  );
}

// 使用方式完全一样
<StudentCard name="张三" age={12} grade="六年级一班" score={95} />
```

## 传递不同类型的数据

### 传递字符串：
```jsx
<Welcome name="小明" />  // 直接写字符串
```

### 传递数字：
```jsx
<Student age={12} score={95} />  // 用大括号包住数字
```

### 传递布尔值：
```jsx
<Product onSale={true} available={false} />
```

### 传递数组：
```jsx
<TodoList tasks={['学习', '吃饭', '睡觉']} />
```

### 传递对象：
```jsx
<UserInfo user={{ name: '小明', age: 12, school: '实验小学' }} />
```

## 实际应用：动态内容

### 场景：消息通知组件
```jsx
function Message({ type, content }) {
  // 根据类型设置不同的样式
  const styles = {
    success: { background: '#d4edda', color: '#155724' },
    error: { background: '#f8d7da', color: '#721c24' },
    warning: { background: '#fff3cd', color: '#856404' }
  };
  
  return (
    <div style={styles[type]}>
      {content}
    </div>
  );
}

// 使用
function App() {
  return (
    <div>
      <Message type="success" content="操作成功！" />
      <Message type="error" content="登录失败，请重试" />
      <Message type="warning" content="您的存储空间不足" />
    </div>
  );
}
```

## 总结

向组件传参很简单：

1. **定义组件时**：用 `props` 或 `{参数名}` 接收
2. **使用组件时**：像 HTML 属性一样传递
3. **传递数据**：字符串直接写，其他类型用 `{ }` 包住
4. **参数命名**：自己起有意义的名称

**记住这个模式：**
```jsx
// 定义
function 组件名({ 参数1, 参数2, 参数3 }) {
  return <div>{参数1} {参数2} {参数3}</div>;
}

// 使用
<组件名 参数1="值1" 参数2={值2} 参数3={值3} />
```
