## 组件确实是函数，但用法不一样

**组件是函数，但你不能直接在 HTML 里写函数名**，而是要用 JSX 语法。

---

## 正确的组件使用方式

### 步骤1：定义组件（写函数）
```jsx
// 这是一个组件（函数）
function Welcome() {
  return <h1>欢迎来到我的网站！</h1>;
}

// 这也是一个组件
function MyButton() {
  return <button>点击我</button>;
}
```

### 步骤2：使用组件（像 HTML 标签）
```jsx
// 在另一个组件中使用它们
function App() {
  return (
    <div>
      <Welcome />   {/* 这样使用组件！ */}
      <MyButton />  {/* 这样使用组件！ */}
    </div>
  );
}
```

**注意：** `<Welcome />` 不是 HTML，是 JSX！

## 完整的工作流程

### 文件结构：
```
src/
  ├── App.js       # 主要组件
  └── index.js     # 入口文件
```

### 1. 在 App.js 中：
```jsx
// 定义子组件
function Welcome() {
  return <h1>你好！</h1>;
}

function MyButton() {
  return <button>点我</button>;
}

// 主组件使用它们
function App() {
  return (
    <div>
      <Welcome />
      <MyButton />
      <Welcome />  {/* 可以重复使用！ */}
    </div>
  );
}

export default App;  // 导出主组件
```

### 2. 在 index.js 中：
```jsx
import React from 'react';
import ReactDOM from 'react-dom';
import App from './App';  // 导入主组件

// 把整个App组件渲染到网页上
ReactDOM.render(<App />, document.getElementById('root'));
```

### 3. 在 public/index.html 中：
```html
<!DOCTYPE html>
<html>
  <head>...</head>
  <body>
    <div id="root"></div>  <!-- React组件会显示在这里 -->
  </body>
</html>
```

## 实际例子：理解组件嵌套

```jsx
// 定义几个小组件
function Header() {
  return <header>网站头部</header>;
}

function Content() {
  return (
    <main>
      <h2>主要内容</h2>
      <p>这里是页面内容...</p>
    </main>
  );
}

function Footer() {
  return <footer>版权信息 © 2024</footer>;
}

// 主组件把所有小组件组合起来
function App() {
  return (
    <div>
      <Header />
      <Content />
      <Footer />
    </div>
  );
}
```

**最终显示效果：**
```
网站头部
主要内容
这里是页面内容...
版权信息 © 2024
```

## 常见错误 vs 正确写法

### ❌ 错误写法：
```jsx
function App() {
  return (
    <div>
      Welcome    {/* 错误！不能直接写函数名 */}
      Welcome()  {/* 错误！不能调用函数 */}
    </div>
  );
}
```

### ✅ 正确写法：
```jsx
function App() {
  return (
    <div>
      <Welcome />    {/* 正确！像HTML标签 */}
      <Welcome></Welcome>  {/* 也正确，但更啰嗦 */}
    </div>
  );
}
```

## 快速练习

你来判断下面哪些是正确的：

```jsx
// 组件定义
function SayHello() {
  return <p>Hello!</p>;
}

function UserInfo() {
  return (
    <div>
      <p>姓名：张三</p>
      <p>年龄：25</p>
    </div>
  );
}

// 使用方法，哪些正确？
function App() {
  return (
    <div>
      {/* 1. */} <SayHello />
      {/* 2. */} SayHello
      {/* 3. */} <UserInfo />
      {/* 4. */} UserInfo()
      {/* 5. */} <SayHello></SayHello>
    </div>
  );
}
```

**答案：** 1、3、5 是正确的！

## 总结

记住这几点：

1. **组件是函数**，但要用 **`<组件名 />`** 的方式使用
2. **组件可以嵌套**：大组件里包含小组件
3. **组件名必须大写开头**：`<Welcome />` ✅，`<welcome />` ❌
4. **组件可以重复使用**：`<Welcome /><Welcome /><Welcome />`

**简单记：组件 = 自定义的HTML标签**

现在明白了吗？组件就是让你可以创建自己的"HTML标签"！😊
