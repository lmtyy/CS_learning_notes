好的！我来详细讲讲如何导入组件，用最简单的例子说明。

## 1. 导入默认导出的组件

### 文件结构：
```
src/
  components/
    Header.jsx
  App.jsx
```

### 导出方（Header.jsx）：
```jsx
// 默认导出一个组件
function Header() {
  return <header>网站标题</header>;
}

export default Header; // 关键：export default
```

### 导入方（App.jsx）：
```jsx
// 方式1：直接导入（推荐）
import Header from './components/Header';

// 方式2：导入并重命名
import SiteHeader from './components/Header';

function App() {
  return (
    <div>
      <Header />        {/* 显示：网站标题 */}
      <SiteHeader />    {/* 显示：网站标题 */}
    </div>
  );
}
```

**特点**：导入时可以任意命名！

## 2. 导入命名导出的组件

### 文件结构：
```
src/
  components/
    Buttons.jsx
  App.jsx
```

### 导出方（Buttons.jsx）：
```jsx
// 命名导出多个组件
export function PrimaryButton() {  // 关键：export
  return <button style={{background: 'blue'}}>主要按钮</button>;
}

export function SecondaryButton() {  // 关键：export
  return <button style={{background: 'gray'}}>次要按钮</button>;
}
```

### 导入方（App.jsx）：
```jsx
// 方式1：导入特定组件
import { PrimaryButton } from './components/Buttons';

// 方式2：导入多个组件
import { PrimaryButton, SecondaryButton } from './components/Buttons';

// 方式3：导入并重命名
import { PrimaryButton as MainBtn } from './components/Buttons';

function App() {
  return (
    <div>
      <PrimaryButton />   {/* 蓝色按钮 */}
      <SecondaryButton /> {/* 灰色按钮 */}
      <MainBtn />         {/* 蓝色按钮 */}
    </div>
  );
}
```

**特点**：必须用 `{}`，必须用原名称（或用 `as` 重命名）

## 3. 混合导入（默认 + 命名）

### 导出方（Layout.jsx）：
```jsx
// 默认导出
export default function Layout() {
  return <div className="layout">布局容器</div>;
}

// 命名导出
export function Sidebar() {
  return <aside>侧边栏</aside>;
}

export function Content() {
  return <main>主要内容</main>;
}
```

### 导入方（App.jsx）：
```jsx
// 同时导入默认和命名导出
import Layout, { Sidebar, Content } from './components/Layout';

function App() {
  return (
    <div>
      <Layout />   {/* 布局容器 */}
      <Sidebar />  {/* 侧边栏 */}
      <Content />  {/* 主要内容 */}
    </div>
  );
}
```

## 4. 实际项目中的路径写法

```jsx
// 从同一目录导入
import Header from './Header';

import Button from './components/Button';
```

## 5. 完整的实际例子

### 文件结构：
```
src/
  components/
    Header.jsx
    Buttons.jsx
  pages/
    Home.jsx
  App.jsx
```

### 各文件内容：
```jsx
// Header.jsx
export default function Header() {
  return <header>我的网站</header>;
}

// Buttons.jsx
export function PrimaryButton({ children }) {
  return <button className="primary">{children}</button>;
}

export function SecondaryButton({ children }) {
  return <button className="secondary">{children}</button>;
}

// Home.jsx
export default function Home() {
  return <div>首页内容</div>;
}

// App.jsx
import Header from './components/Header';
import Home from './pages/Home';
import { PrimaryButton, SecondaryButton } from './components/Buttons';

function App() {
  return (
    <div>
      <Header />
      <Home />
      <PrimaryButton>登录</PrimaryButton>
      <SecondaryButton>注册</SecondaryButton>
    </div>
  );
}

export default App;
```

## 记忆口诀：

1. **默认导出**：`import 任意名字 from '路径'`
2. **命名导出**：`import { 原名 } from '路径'`
3. **混合导入**：`import 默认, { 命名1, 命名2 } from '路径'`

这样应该很清楚了吧！实际开发中多练习几次就熟悉了。
