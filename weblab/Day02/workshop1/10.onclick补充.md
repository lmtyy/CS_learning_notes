不是的！`onClick` 在很多元素上都可以用。

## 1. 在 div 上使用

```jsx
function App() {
  const handleClick = () => {
    alert('div被点击了！');
  };

  return (
    <div 
      onClick={handleClick}
      style={{
        padding: '20px',
        background: 'lightblue',
        margin: '10px'
      }}
    >
      点击这个div区域
    </div>
  );
}
```

## 2. 在图片上使用

```jsx
function App() {
  const handleClick = () => {
    alert('图片被点击了！');
  };

  return (
    <img 
      src="/path/to/image.jpg" 
      alt="示例图片"
      onClick={handleClick}
      style={{ cursor: 'pointer' }}
    />
  );
}
```

## 3. 在文字上使用

```jsx
function App() {
  const handleClick = () => {
    alert('文字被点击了！');
  };

  return (
    <p onClick={handleClick}>
      点击这段文字
    </p>
  );
}
```

## 4. 在列表项上使用

```jsx
function App() {
  const handleClick = () => {
    alert('列表项被点击了！');
  };

  return (
    <ul>
      <li onClick={handleClick}>第一项</li>
      <li onClick={handleClick}>第二项</li>
      <li onClick={handleClick}>第三项</li>
    </ul>
  );
}
```

## 5. 在标题上使用

```jsx
function App() {
  const handleClick = () => {
    alert('标题被点击了！');
  };

  return (
    <h1 onClick={handleClick}>
      点击这个标题
    </h1>

}
```

## 6. 实际例子：点击切换颜色

```jsx
function App() {
  const [isActive, setIsActive] = useState(false);
  
  const handleClick = () => {
    setIsActive(!isActive);
  };

  return (
    <div 
      onClick={handleClick}
      style={{
        padding: '20px',
        background: isActive ? 'red' : 'blue',
        color: 'white'
      }}
    >
      点击我改变颜色
    </div>
  );
}
```

## 总结

`onClick` 可以在**几乎所有可见的HTML元素**上使用：
- `button`、`div`、`p`、`span`
- `img`、`li`、`h1`-`h6`
- 等等...

**注意**：为了让用户知道可以点击，最好加上 `style={{ cursor: 'pointer' }}`，这样鼠标会变成手型。
