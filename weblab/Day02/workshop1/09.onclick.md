## 1. ==什么是 onClick？==

`onClick` 就是"当点击时"的意思。比如：

```jsx
<button onClick={这里写点击后要做什么}>
  点击我
</button>
```

## 2. 最简单的例子

```jsx
function App() {
  // 点击按钮时弹出提示
  const handleClick = () => {
    alert('你好！');
  };

  return (
    <button onClick={handleClick}>
      点击我
    </button>
  );
}
```

效果：点击按钮 → 弹出"你好！"

## 3. 另一种写法

也可以直接写在按钮里：

```jsx
function App() {
  return (
    <button onClick={() => alert('你好！')}>
      点击我
    </button>
  );
}
```

效果是一样的。

## 4. 为什么要这样写？

在 HTML 中是这样写的：
```html
<button onclick="handleClick()">点击</button>
```

**但在 React 中**：
- 要用 `onClick`（C 大写）
- 要用 `{}` 包住
- 直接写函数名，不加 `()`

## 5. 改变文字的例子

```jsx
function App() {
  const [text, setText] = useState('原始文字');
  
  const changeText = () => {
    setText('新文字！');
  };

  return (
    <div>
      <p>{text}</p>
      <button onClick={changeText}>
        改变文字
      </button>
    </div>
  );
}
```

效果：点击按钮 → 文字从"原始文字"变成"新文字！"

## 6. 计数器的例子

```jsx
function App() {
  const [count, setCount] = useState(0);
  
  const addOne = () => {
    setCount(count + 1);
  };

  return (
    <div>
      <p>当前计数：{count}</p>
      <button onClick={addOne}>
        加1
      </button>
    </div>
  );
}
```

效果：点击按钮 → 数字不断增加

## 总结

`onClick` 就是：
1. **==写在标签里==**：`<button onClick={...}>`
2. **==里面写函数==**：点击后要执行的代码
3. **==改变页面==**：通过改变 state 来更新显示
