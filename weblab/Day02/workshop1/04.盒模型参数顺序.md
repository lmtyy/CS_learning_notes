在 CSS 中，盒模型相关的简写属性有特定的参数顺序：

## 1. `margin` 和 `padding`
**==顺序：上 → 右 → 下 → 左==**（顺时针）

```css
/* 四个值：上 右 下 左 */
margin: 10px 20px 30px 40px;
/* 相当于： */
margin-top: 10px;
margin-right: 20px;
margin-bottom: 30px;
margin-left: 40px;

/* 三个值：上 左右 下 */
margin: 10px 20px 30px;
/* 相当于： */
margin-top: 10px;
margin-right: 20px;
margin-left: 20px;
margin-bottom: 30px;

/* 两个值：上下 左右 */
margin: 10px 20px;
/* 相当于： */
margin-top: 10px;
margin-bottom: 10px;
margin-right: 20px;
margin-left: 20px;

/* 一个值：所有边 */
margin: 10px;
/* 相当于： */
margin-top: 10px;
margin-right: 10px;
margin-bottom: 10px;
margin-left: 10px;
```

## 2. `border`
**顺序：宽度 样式 颜色**

```css
border: 2px solid #000;
/* 相当于： */
border-width: 2px;
border-style: solid;
border-color: #000;
```

## 3. `border-radius`
**顺序：左上 右上 右下 左下**（顺时针）

```css
/* 四个值：左上 右上 右下 左下 */
border-radius: 10px 20px 30px 40px;

/* 三个值：左上 右上/左下 右下 */
border-radius: 10px 20px 30px;

/* 两个值：左上/右下 右上/左下 */
border-radius: 10px 20px;

/* 一个值：所有角 */
border-radius: 10px;
```

## 4. `box-shadow`
**顺序：水平偏移 垂直偏移 模糊半径 扩展半径 颜色 内阴影**

```css
box-shadow: 2px 2px 5px 1px rgba(0,0,0,0.3);
/* 相当于： */
box-shadow: h-offset v-offset blur spread color inset;
```

## 记忆技巧：
- **边距类**：顺时针方向（上→右→下→左）
- **边框类**：宽度→样式→颜色
- **圆角类**：从左上角开始顺时针
- **阴影类**：水平→垂直→模糊→扩展→颜色→内阴影

这些顺序是固定的，不能随意调换！
