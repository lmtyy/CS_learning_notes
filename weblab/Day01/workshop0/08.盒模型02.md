# CSS 盒模型简单示例

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>CSS盒模型简单示例</title>
    <style>
        /* 基础样式 */
        body {
            font-family: Arial, sans-serif;
            padding: 20px;
            background-color: #f5f5f5;
        }
        
        .container {
            max-width: 600px;
            margin: 0 auto;
        }
        
        /* 盒模型演示 */
        .box {
            background-color: #4CAF50;
            color: white;
            text-align: center;
            padding: 20px;
            border: 5px solid #FF5722;
            margin: 30px;
            width: 200px;
        }
        
        /* 可视化辅助线 */
        .margin-area {
            background-color: #FFE0B2;
            padding: 30px;
            border: 2px dashed #FF9800;
        }
        
        .border-area {
            background-color: #FFF3E0;
            padding: 5px;
        }
        
        .padding-area {
            background-color: #E8F5E8;
            padding: 20px;
        }
        
        .content-area {
            background-color: #4CAF50;
            color: white;
            padding: 20px;
            height: 60px;
            display: flex;
            align-items: center;
            justify-content: center;
        }
        
        /* 说明文字 */
        .explanation {
            margin-top: 30px;
            padding: 15px;
            background: white;
            border-radius: 5px;
            box-shadow: 0 2px 5px rgba(0,0,0,0.1);
        }
        
        .dimension {
            font-weight: bold;
            color: #FF5722;
        }
    </style>
</head>
<body>
    <div class="container">
        <h1>CSS盒模型可视化示例</h1>
        
        <!-- 盒模型可视化 -->
        <div class="margin-area">
            <div class="margin-label">外边距 (Margin) - 30px</div>
            <div class="border-area">
                <div class="border-label">边框 (Border) - 5px</div>
                <div class="padding-area">
                    <div class="padding-label">内边距 (Padding) - 20px</div>
                    <div class="content-area">
                        内容区域 (Content)<br>
                        宽度: <span class="dimension">200px</span>
                    </div>
                </div>
            </div>
        </div>
        
        <!-- 计算说明 -->
        <div class="explanation">
            <h3>盒模型计算：</h3>
            <p><strong>元素总宽度 = width + padding + border</strong></p>
            <p>200px (内容) + 40px (左右padding) + 10px (左右border) = <span class="dimension">250px</span></p>
            <p><strong>页面占据宽度 = 总宽度 + margin</strong></p>
            <p>250px + 60px (左右margin) = <span class="dimension">310px</span></p>
        </div>
        
        <!-- 对比示例 -->
        <div class="explanation">
            <h3>box-sizing 对比：</h3>
            <div style="display: flex; gap: 20px; margin-top: 15px;">
                <div style="flex: 1; padding: 15px; background: #E3F2FD; border-radius: 5px;">
                    <h4>content-box (默认)</h4>
                    <p>width: 200px</p>
                    <p>padding: 20px</p>
                    <p>border: 5px</p>
                    <p><strong>实际宽度: 250px</strong></p>
                </div>
                <div style="flex: 1; padding: 15px; background: #E8F5E9; border-radius: 5px;">
                    <h4>border-box</h4>
                    <p>width: 200px</p>
                    <p>padding: 20px</p>
                    <p>border: 5px</p>
                    <p><strong>实际宽度: 200px</strong></p>
                </div>
            </div>
        </div>
    </div>
</body>
</html>
```

## 核心概念解释：

### 1. **四个层次（从内到外）：**
- **Content（内容）**：文字、图片等实际内容
- **Padding（内边距）**：内容与边框之间的空间
- **Border（边框）**：围绕内容和内边距的线
- **Margin（外边距）**：元素与其他元素之间的空间

### 2. **重要计算：**
```css
.box {
    width: 200px;        /* 内容宽度 */
    padding: 20px;       /* 上下左右内边距 */
    border: 5px solid;   /* 边框宽度 */
    margin: 30px;        /* 上下左右外边距 */
}
```

**总宽度计算：**
- 内容宽度: 200px
- 左右内边距: 20px + 20px = 40px  
- 左右边框: 5px + 5px = 10px
- **元素总宽度: 200 + 40 + 10 = 250px**
- 左右外边距: 30px + 30px = 60px
- **页面占据宽度: 250 + 60 = 310px**

### 3. **box-sizing 的区别：**
- **content-box**（默认）：width 只包含内容
- **border-box**：width 包含内容 + padding + border

这个示例直观地展示了盒模型的每个部分，你可以直接复制代码运行查看效果！
