# 8pt Grid System (8点网格系统) 详解

## 核心概念

**8pt Grid System** 是一种设计系统，所有尺寸（间距、边距、大小等）都使用 8 的倍数作为基本单位。

## 基本原理

```css
/* 使用 8 的倍数作为所有尺寸单位 */
:root {
  --space-1: 8px;   /* 8 × 1 */
  --space-2: 16px;  /* 8 × 2 */
  --space-3: 24px;  /* 8 × 3 */
  --space-4: 32px;  /* 8 × 4 */
  --space-5: 40px;  /* 8 × 5 */
  --space-6: 48px;  /* 8 × 6 */
}

.container {
  padding: var(--space-3);        /* 24px */
  margin-bottom: var(--space-4);  /* 32px */
}

.button {
  padding: var(--space-2) var(--space-3);  /* 16px 24px */
  height: var(--space-6);                  /* 48px */
}
```

## 为什么选择 8？

### 1. **数学优势**
- 8 可以被 2、4 整除，易于缩放
- 在响应式设计中容易计算比例
- 大多数屏幕尺寸能被 8 整除

### 2. **实际例子对比**

```css
/* ❌ 不一致的间距 */
.element {
  margin: 13px;
  padding: 7px;
  gap: 19px;
}

/* ✅ 8pt 网格系统 */
.element {
  margin: 16px;    /* 8 × 2 */
  padding: 8px;    /* 8 × 1 */
  gap: 24px;       /* 8 × 3 */
}
```

## 完整的 8pt 尺度系统

```css
/* 间距尺度 */
:root {
  /* 基础单位 */
  --base-unit: 8px;
  
  /* 小尺寸 */
  --space-xs: 4px;   /* 8 × 0.5 */
  --space-sm: 8px;   /* 8 × 1 */
  --space-md: 16px;  /* 8 × 2 */
  --space-lg: 24px;  /* 8 × 3 */
  --space-xl: 32px;  /* 8 × 4 */
  --space-2xl: 40px; /* 8 × 5 */
  --space-3xl: 48px; /* 8 × 6 */
}

/* 组件高度 */
:root {
  --height-sm: 32px;  /* 8 × 4 */
  --height-md: 40px;  /* 8 × 5 */
  --height-lg: 48px;  /* 8 × 6 */
}
```

## 实际应用示例

```css
/* 卡片组件 */
.card {
  padding: var(--space-lg);     /* 24px */
  margin-bottom: var(--space-md); /* 16px */
  border-radius: var(--space-sm); /* 8px */
}

/* 按钮组件 */
.btn {
  padding: var(--space-sm) var(--space-lg); /* 8px 24px */
  height: var(--height-md);                 /* 40px */
  gap: var(--space-sm);                     /* 8px */
}

/* 布局系统 */
.container {
  max-width: 1200px;  /* 8 × 150 */
  padding: var(--space-lg);
  gap: var(--space-xl); /* 32px */
}

.grid {
  gap: var(--space-md); /* 16px */
  margin: var(--space-xl) 0; /* 32px */
}
```

## 响应式设计中的应用

```css
/* 移动端使用较小的基数 */
@media (max-width: 768px) {
  :root {
    --base-unit: 4px;  /* 在移动端使用 4pt 系统 */
  }
}

/* 或者保持 8pt 但使用不同的尺度 */
.mobile-card {
  padding: 16px;  /* 在移动端仍然使用 8 的倍数 */
  margin: 8px;
}
```

## 主要优势

1. **一致性**：整个产品保持统一的视觉节奏
2. **开发效率**：减少决策时间，直接选择预设尺寸
3. **响应式友好**：容易计算比例和缩放
4. **团队协作**：设计师和开发者使用相同的尺度系统

## 现代 CSS 的实现方式

```css
/* 使用 CSS 自定义属性 */
:root {
  --space-scale: 0.5rem; /* 8px 如果 1rem = 16px */
  
  --space-1: calc(var(--space-scale) * 1); /* 8px */
  --space-2: calc(var(--space-scale) * 2); /* 16px */
  --space-3: calc(var(--space-scale) * 3); /* 24px */
}

/* 使用 CSS Grid 和 Flexbox */
.grid-layout {
  display: grid;
  gap: var(--space-3); /* 24px */
  padding: var(--space-2); /* 16px */
}
```

**总结：** 8pt Grid System 通过使用 8 的倍数作为所有尺寸的基础单位，创建了统一、协调的视觉层次，提高了设计的一致性和开发效率。
