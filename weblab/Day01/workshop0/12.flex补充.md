# Flex-grow 和 Flex-basis 详解

## Flex-grow（放大比例）

### 作用
定义项目在容器有**剩余空间**时的放大比例

### 取值
- `0`（默认）：不放大
- `正整数`：按比例分配剩余空间

### 示例
```css
.item1 { flex-grow: 1; }  /* 占1份 */
.item2 { flex-grow: 2; }  /* 占2份 */
.item3 { flex-grow: 1; }  /* 占1份 */
```
剩余空间按 1:2:1 的比例分配

### 计算方式
```
项目最终尺寸 = 项目原本尺寸 + (剩余空间 × flex-grow / 总grow值)
```

---

## Flex-basis（基础尺寸）

### 作用
定义项目在分配多余空间之前的**初始尺寸**

### 取值
- `auto`（默认）：项目的本来大小
- `长度值`：固定尺寸（px, %, rem等）
- `content`：根据内容自动计算

### 示例
```css
.item1 { flex-basis: 200px; }  /* 固定200px */
.item2 { flex-basis: 50%; }    /* 容器宽度的50% */
.item3 { flex-basis: auto; }   /* 内容宽度 */
```

---

## 实际应用场景

### 1. 等分布局
```css
.item {
  flex: 1; /* 等价于 flex: 1 1 0% */
}
/* 所有项目等分容器宽度 */
```

### 2. 固定+自适应布局
```css
.sidebar {
  flex: 0 0 200px; /* 不放大不缩小，固定200px */
}
.content {
  flex: 1; /* 占据剩余所有空间 */
}
```

### 3. 比例布局
```css
.item1 { flex: 2; } /* 占2份 */
.item2 { flex: 1; } /* 占1份 */
.item3 { flex: 1; } /* 占1份 */
/* 按2:1:1的比例分配空间 */
```

### 4. 内容自适应
```css
.item {
  flex: 0 1 auto; /* 不放大，可缩小，基于内容宽度 */
}
```

---

## 核心理解

- **flex-grow**：处理**剩余空间**的分配
- **flex-basis**：设置**初始尺寸**，影响剩余空间的计算
- 两者配合使用可以创建各种灵活的布局模式

记住：flex-grow只在有剩余空间时生效，flex-basis决定了项目的"起点尺寸"。
