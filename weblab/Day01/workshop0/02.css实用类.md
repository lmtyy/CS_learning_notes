# 实用类(Utility Classes)详解

实用类(Utility Classes)是一种CSS设计方法，专注于创建单一用途、可重用的类，通常用于快速应用特定的样式规则。

## 实用类的关键特点

1. **单一职责**：每个实用类只负责一个特定的样式属性
2. **可重用性**：可以在整个项目中重复使用
3. **命名约定**：通常使用前缀（如"u-"）来标识
4. **组合使用**：通过组合多个实用类来创建复杂样式

## 实用类框架

这种方法也是现代CSS框架（如Tailwind CSS）的核心思想，它们提供了大量预定义的实用类来加速开发。
