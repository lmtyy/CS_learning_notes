**必须使用 `var()` 函数来调用 CSS 变量。**

## 基本语法

```css
/* 定义变量 */
:root {
    --primary-color: #396dff;
    --spacing: 20px;
}

/* 使用变量 - 正确 ✅ */
.button {
    background-color: var(--primary-color);
    padding: var(--spacing);
}

/* 使用变量 - 错误 ❌ */
.button {
    background-color: --primary-color; /* 这样不行！ */
    padding: --spacing; /* 这样也不行！ */
}
```

## `var()` 函数的完整用法

```css
:root {
    --main-color: #396dff;
    --backup-color: #ff0000;
}

.element {
    /* 基本用法 */
    color: var(--main-color);
    
    /* 带备用值的用法 */
    color: var(--main-color, blue); /* 如果 --main-color 不存在，使用 blue */
    
    /* 多层备用值 */
    color: var(--undefined-var, var(--backup-color, green));
    /* 先尝试 --undefined-var，再尝试 --backup-color，最后用 green */
}
```

## 实际例子

```css
:root {
    --primary: #396dff;
    --danger: #ff4757;
    --padding-sm: 8px;
    --padding-md: 16px;
}

.btn-primary {
    background: var(--primary);
    padding: var(--padding-sm) var(--padding-md);
}

.btn-danger {
    background: var(--danger);
    padding: var(--padding-sm) var(--padding-md);
}

/* 如果要修改主题色，只需要改这一个地方 */
:root {
    --primary: #ff6b6b; /* 所有使用 --primary 的地方都会自动更新 */
}
```

**总结：定义变量用 `--变量名`，使用变量必须用 `var(--变量名)`。**
