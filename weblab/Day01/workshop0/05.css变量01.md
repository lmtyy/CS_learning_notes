# CSS 变量（自定义属性）详解

CSS 变量（也称为自定义属性）是 CSS 中强大的功能，允许您定义可重用的值并在整个样式表中使用它们。

## 基本概念

```css
/* 定义 CSS 变量 */
:root {
    --primary-color: #396dff;
    --grey-color: #f7f7f7;
    --white-color: #ffffff;
    --spacing: 16px;
    --border-radius: 8px;
}

/* 使用 CSS 变量 */
.button {
    background-color: var(--primary-color);
    color: var(--white-color);
    padding: var(--spacing);
    border-radius: var(--border-radius);
}
```

## 关键知识点

### 1. **变量命名**
- 必须以 `--` 开头
- 区分大小写：`--color` 和 `--Color` 是不同的变量
- 推荐使用 kebab-case 命名法：`--primary-color`

### 2. **作用域**
- `:root` 伪类中定义的变量是全局变量
- 也可以在特定选择器中定义局部变量

```css
/* 全局变量 */
:root {
    --main-color: blue;
}

/* 局部变量 - 只在 .container 内有效 */
.container {
    --container-bg: #f0f0f0;
    background: var(--container-bg);
}
```

### 3. **使用 var() 函数**
```css
.element {
    color: var(--primary-color);
    margin: var(--spacing, 10px); /* 第二个参数是备用值 */
}
```

## 实际优势

1. **维护性**：修改一个变量值，所有使用的地方自动更新
2. **一致性**：确保整个网站使用相同的颜色、间距等
3. **主题切换**：轻松实现明暗主题切换
4. **动态计算**：可以与 calc() 函数结合使用

## 实际应用示例

```css
:root {
    --primary: #396dff;
    --secondary: #6c757d;
    --success: #28a745;
    --spacing-sm: 8px;
    --spacing-md: 16px;
    --spacing-lg: 24px;
}

.theme-dark {
    --bg-color: #1a1a1a;
    --text-color: #ffffff;
}

.theme-light {
    --bg-color: #ffffff;
    --text-color: #333333;
}

body {
    background-color: var(--bg-color);
    color: var(--text-color);
    padding: var(--spacing-md);
}
```

CSS 变量让样式表更加模块化、可维护，是现代 CSS 开发的重要工具。
