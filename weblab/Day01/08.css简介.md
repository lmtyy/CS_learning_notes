好的！CSS（层叠样式表）是为 HTML 添加样式的语言，让网页从简陋变得美观。让我为你系统讲解 CSS 的基本知识。

## 1. CSS 的基本语法

### 选择器 + 声明块
```css
选择器 {
    属性: 值;
    属性: 值;
}
```

### 实际例子
```css
h1 {
    color: blue;
    font-size: 24px;
    text-align: center;
}
```

**解释：**
- `h1` - 选择器（选中所有 `<h1>` 标签）
- `color: blue;` - 声明（属性: 值）
- `{}` - 声明块（包含所有样式声明）

## 2. 如何将 CSS 应用到 HTML

### 方法1：外部样式表（最推荐）
```html
<!DOCTYPE html>
<html>
<head>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <h1>我的标题</h1>
</body>
</html>
```

### 方法2：内部样式表
```html
<!DOCTYPE html>
<html>
<head>
    <style>
        h1 {
            color: blue;
            font-size: 24px;
        }
    </style>
</head>
<body>
    <h1>我的标题</h1>
</body>
</html>
```

### 方法3：行内样式（不推荐）
```html
<h1 style="color: blue; font-size: 24px;">我的标题</h1>
```

## 3. 基本选择器

### 元素选择器
```css
/* 选中所有 <p> 标签 */
p {
    color: black;
}

/* 选中所有 <h1> 标签 */
h1 {
    font-size: 32px;
}
```

### 类选择器（最常用）
```html
<!-- HTML -->
<p class="highlight">重要内容</p>
<div class="card warning">警告卡片</div>
```

```css
/* CSS - 以点(.)开头 */
.highlight {
    background-color: yellow;
}

.card {
    border: 1px solid #ccc;
    padding: 20px;
}

.warning {
    border-color: red;
    color: red;
}
```

### ID 选择器
```html
<!-- HTML -->
<div id="header">页面头部</div>
```

```css
/* CSS - 以井号(#)开头 */
#header {
    background-color: #333;
    color: white;
    padding: 20px;
}
```

### 通用选择器
```css
/* 选中页面所有元素 */
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}
```

## 4. 颜色和单位

### 颜色表示方法
```css
p {
    color: red;                    /* 颜色名称 */
    color: #ff0000;               /* 十六进制 */
    color: rgb(255, 0, 0);        /* RGB */
    color: rgba(255, 0, 0, 0.5);  /* RGB + 透明度 */
    color: hsl(0, 100%, 50%);     /* 色相饱和度亮度 */
}
```

### 常用单位
```css
div {
    width: 100px;          /* 像素 */
    width: 50%;            /* 百分比 */
    font-size: 1.5em;      /* 相对于父元素 */
    font-size: 1.5rem;     /* 相对于根元素 */
    line-height: 1.6;      /* 无单位，倍数 */
    margin: 10px 20px;     /* 多个值 */
}
```

## 5. 文本样式

```css
p {
    /* 字体相关 */
    font-family: Arial, sans-serif;  /* 字体族 */
    font-size: 16px;                 /* 字号 */
    font-weight: bold;               /* 字重 */
    font-style: italic;              /* 斜体 */
    
    /* 文本相关 */
    color: #333;                     /* 颜色 */
    text-align: center;              /* 对齐 */
    line-height: 1.6;                /* 行高 */
    text-decoration: underline;      /* 下划线 */
    text-transform: uppercase;       /* 大小写 */
    letter-spacing: 1px;             /* 字间距 */
}
```

## 6. 盒模型（核心概念）

每个 HTML 元素都是一个盒子，由四部分组成：

```css
div {
    width: 300px;        /* 内容宽度 */
    height: 200px;       /* 内容高度 */
    padding: 20px;       /* 内边距 */
    border: 2px solid black;  /* 边框 */
    margin: 30px;        /* 外边距 */
}
```

### 盒模型可视化
```
┌─────────────────────────────────┐
│            Margin (30px)        │
│  ┌─────────────────────────┐    │
│  │      Border (2px)       │    │
│  │  ┌─────────────────┐    │    │
│  │  │   Padding (20px)│    │    │
│  │  │  ┌─────────┐    │    │    │
│  │  │  │ Content │    │    │    │
│  │  │  │ 300x200 │    │    │    │
│  │  │  └─────────┘    │    │    │
│  │  └─────────────────┘    │    │
│  └─────────────────────────┘    │
└─────────────────────────────────┘
```

### 简写写法
```css
div {
    /* 四个方向相同 */
    margin: 10px;
    padding: 10px;
    
    /* 上下 | 左右 */
    margin: 10px 20px;
    padding: 10px 20px;
    
    /* 上 | 右 | 下 | 左 (顺时针) */
    margin: 10px 20px 30px 40px;
    padding: 10px 20px 30px 40px;
    
    /* 边框简写 */
    border: 2px solid red;
    /* 等价于：
    border-width: 2px;
    border-style: solid;
    border-color: red;
    */
}
```

## 7. 常用布局属性

### display 属性
```css
.block {
    display: block;      /* 块级元素，独占一行 */
}

.inline {
    display: inline;     /* 行内元素，不换行 */
}

.inline-block {
    display: inline-block; /* 行内块，可设宽高 */
}

.none {
    display: none;       /* 完全隐藏 */
}
```

### 定位 position
```css
.relative {
    position: relative;  /* 相对定位 */
    top: 10px;
    left: 20px;
}

.absolute {
    position: absolute;  /* 绝对定位 */
    top: 0;
    right: 0;
}

.fixed {
    position: fixed;     /* 固定定位 */
    bottom: 20px;
    right: 20px;
}

.sticky {
    position: sticky;    /* 粘性定位 */
    top: 0;
}
```

## 9. 响应式设计基础

### 媒体查询
```css
/* 默认样式（手机） */
.container {
    width: 100%;
    padding: 10px;
}

/* 平板电脑 */
@media (min-width: 768px) {
    .container {
        width: 750px;
        margin: 0 auto;
    }
}

/* 桌面电脑 */
@media (min-width: 1024px) {
    .container {
        width: 980px;
    }
}
```

## 学习建议

1. **从基础开始** - 先掌握选择器、盒模型、常用属性
2. **多动手实践** - 边学边写代码，看到实际效果
3. **理解盒模型** - 这是 CSS 布局的核心
4. **学习 Flexbox** - 现代布局的必备技能
5. **使用开发者工具** - 调试和学习的利器

CSS 的核心思想就是：**通过选择器选中 HTML 元素，然后为它们添加样式**。掌握了这些基础知识，你就能够创建美观的网页了！
