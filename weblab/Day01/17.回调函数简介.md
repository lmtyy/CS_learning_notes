你提供的图片内容是在讲解 **JavaScript 中的回调函数（Callback Functions）**。

我来用简单的中文解释一下这段话的意思：

---

### 核心概念

**"在 JavaScript 中，函数可以像其他值一样被传递"**

这意味着：
- 函数和数字、字符串、对象一样，都是一种**值**
- 所以函数可以：
  - 被赋值给变量
  - 作为参数传递给其他函数
  - 从函数中返回

**"我们可以把一个回调函数作为参数传给另一个函数！"**

这就是**回调函数**的定义：
- 你把一个函数A作为参数，传给另一个函数B
- 然后函数B可以在适当的时机**调用**你传给它的这个函数A
- 因为这个函数A是"被回过头来调用"的，所以叫"回调函数"

---

### 简单例子

```javascript
// 1. 先定义一个回调函数
function sayHello() {
    console.log("Hello!");
}

// 2. 把回调函数传给另一个函数
function doSomething(callback) {
    console.log("我先做点别的事情...");
    
    // 在适当的时候"回调"传进来的函数
    callback(); // 这里调用了sayHello函数
}

// 3. 使用：把sayHello作为回调函数传递
doSomething(sayHello); 

// 输出：
// "我先做点别的事情..."
// "Hello!"
```

---

### 更常见的例子：定时器

```javascript
// setTimeout 就接受一个回调函数
setTimeout(function() {
    console.log("3秒后执行这个回调函数！");
}, 3000);
```

这里：
- 我们创建了一个匿名函数作为回调
- 把这个回调函数传给了 `setTimeout`
- `setTimeout` 会在3秒后"回过头来"执行我们这个函数

---

### 总结

**回调函数就是：**
1. **把一个函数作为参数传给另一个函数**
2. **接收函数在需要的时候会"回调"这个传进来的函数**

这是 JavaScript 中非常重要的概念，是理解异步编程、事件处理等高级特性的基础！
