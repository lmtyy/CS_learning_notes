## 1. 文档结构标签

这些标签定义了 HTML 文档的基本框架。

```html
<!DOCTYPE html>
<html>
    <head>
        <title>页面标题</title>
    </head>
    <body>
        <!-- 页面内容放在这里 -->
    </body>
</html>
```

- `<!DOCTYPE html>` - 声明文档类型为 HTML5
- `<html>` - 根元素，包裹整个页面
- `<head>` - 包含元信息，如标题、字符集、样式表链接等
- `<title>` - 定义浏览器标签页上显示的标题
- `<body>` - 包含所有可见的页面内容

## 2. 文本内容标签

### 标题标签
```html
<h1>一级标题 - 最重要的标题</h1>
<h2>二级标题</h2>
<h3>三级标题</h3>
<h4>四级标题</h4>
<h5>五级标题</h5>
<h6>六级标题 - 最不重要的标题</h6>
```

### 段落和文本格式
```html
<p>这是一个段落。段落会自动换行并有一定的外边距。</p>

<p>
    <strong>加粗文本（有语义重要性）</strong>，
    <b>加粗文本（无语义）</b>，
    <em>强调文本（斜体，有语义）</em>，
    <i>斜体文本（无语义）</i>，
    <u>下划线文本</u>，
    <s>删除线文本</s>
</p>

<hr> <!-- 水平分割线 -->
<br> <!-- 换行 -->
```

## 3. 列表标签

### 无序列表
```html
<ul>
    <li>列表项 1</li>
    <li>列表项 2</li>
    <li>列表项 3</li>
</ul>
```

### 有序列表
```html
<ol>
    <li>第一步</li>
    <li>第二步</li>
    <li>第三步</li>
</ol>
```

### 定义列表
```html
<dl>
    <dt>HTML</dt>
    <dd>超文本标记语言</dd>
    
    <dt>CSS</dt>
    <dd>层叠样式表</dd>
</dl>
```

## 4. 链接和图片标签

```html
<!-- 链接 -->
<a href="https://www.example.com">访问示例网站</a>
<a href="about.html">关于我们</a>
<a href="#section1">跳转到第一节</a>

<!-- 图片 -->
<img src="image.jpg" alt="图片描述" width="400" height="300">
```

- `href` - 指定链接地址
- `src` - 指定图片源
- `alt` - 图片无法显示时的替代文本（重要！）

## 7. 表单标签

表单用于收集用户输入。

```html
<form action="/submit" method="POST">
    <!-- 文本输入 -->
    <label for="username">用户名：</label>
    <input type="text" id="username" name="username" placeholder="请输入用户名">
    
    <br>
    
    <!-- 密码输入 -->
    <label for="password">密码：</label>
    <input type="password" id="password" name="password">
    
    <br>
    
    <!-- 邮箱输入 -->
    <label for="email">邮箱：</label>
    <input type="email" id="email" name="email">
    
    <br>
    
    <!-- 单选按钮 -->
    <label>性别：</label>
    <input type="radio" id="male" name="gender" value="male">
    <label for="male">男</label>
    <input type="radio" id="female" name="gender" value="female">
    <label for="female">女</label>
    
    <br>
    
    <!-- 复选框 -->
    <label>兴趣：</label>
    <input type="checkbox" id="sports" name="interests" value="sports">
    <label for="sports">运动</label>
    <input type="checkbox" id="music" name="interests" value="music">
    <label for="music">音乐</label>
    
    <br>
    
    <!-- 下拉选择 -->
    <label for="city">城市：</label>
    <select id="city" name="city">
        <option value="beijing">北京</option>
        <option value="shanghai">上海</option>
        <option value="guangzhou">广州</option>
    </select>
    
    <br>
    
    <!-- 文本域 -->
    <label for="message">留言：</label>
    <textarea id="message" name="message" rows="4" cols="50"></textarea>
    
    <br>
    
    <!-- 按钮 -->
    <button type="submit">提交</button>
    <button type="reset">重置</button>
</form>
```

## 学习建议

1. **从语义化开始** - 尽量使用有意义的标签
2. **理解嵌套关系** - 正确嵌套标签（如列表项必须在列表内）
3. **掌握常用属性** - 如 `id`, `class`, `src`, `href`, `alt` 等
4. **实践练习** - 多写代码，尝试组合不同的标签

这些基础标签足以让你构建出结构良好、语义清晰的网页！
