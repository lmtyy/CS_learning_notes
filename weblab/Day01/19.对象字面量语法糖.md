这个 `{ product, price }` 是 **ES6 的对象属性简写语法**。

让我通过对比来解释：

## 传统写法（ES5）
```javascript
addItem(product, price) {
  this.items.push({
    product: product,
    price: price
  });
}
```

## 简写写法（ES6）
```javascript
addItem(product, price) {
  this.items.push({ product, price });
}
```

---

## 两种写法完全等价！

### 工作原理：
当**属性名和变量名相同**时，可以省略冒号和值：

```javascript
const name = "Alice";
const age = 25;

// 传统写法
const person1 = {
  name: name,
  age: age
};

// 简写写法
const person2 = { name, age };

console.log(person1); // { name: "Alice", age: 25 }
console.log(person2); // { name: "Alice", age: 25 } - 结果完全相同！
```

---

## 更多例子

### 例1：多个属性
```javascript
const x = 10;
const y = 20;
const z = 30;

// 简写
const point = { x, y, z };
// 等价于：{ x: x, y: y, z: z }
```

### 例2：混合使用
```javascript
const width = 100;
const height = 200;

// 可以简写和传统写法混用
const rectangle = {
  width,           // 简写
  height,          // 简写  
  color: "blue",   // 传统写法（属性名和变量名不同）
  area: function() { return width * height; }  // 方法
};
```

---

## 在函数返回中特别有用

```javascript
function createUser(name, email) {
  // 传统写法
  // return { name: name, email: email };
  
  // 简写写法 - 更简洁！
  return { name, email };
}

const user = createUser("Alice", "alice@example.com");
console.log(user); // { name: "Alice", email: "alice@example.com" }
```

---

## 总结

`{ product, price }` 是 ES6 的语法糖，它相当于：
```javascript
{
  product: product,
  price: price
}
```

**使用条件**：当对象的属性名与要赋值的变量名相同时，就可以使用这种简写语法，让代码更简洁清晰！

这是现代 JavaScript 中非常常见和推荐的写法。
