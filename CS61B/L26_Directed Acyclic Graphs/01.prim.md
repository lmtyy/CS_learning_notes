# Prim 算法详解（Java 实现）

## 算法概述

Prim 算法是一种用于求解**加权无向图的最小生成树（MST）** 的贪心算法。最小生成树是指连接图中所有顶点的边构成的树，且所有边的权重之和最小。

## 算法思想

1. 从任意一个顶点开始，初始化一个只包含该顶点的集合
2. 每次选择连接已选顶点集合和未选顶点集合的最小权重边
3. 将该边连接的顶点加入已选集合
4. 重复步骤2-3，直到所有顶点都被包含

## 时间复杂度

- **邻接矩阵**：O(V²)
- **二叉堆 + 邻接表**：O(E log V)
- **斐波那契堆 + 邻接表**：O(E + V log V)

## Java 实现

### 方法一：使用邻接矩阵（适合稠密图）

```java
import java.util.*;

public class PrimAlgorithm {
    
    /**
     * 使用邻接矩阵实现Prim算法
     * @param graph 邻接矩阵表示的图
     * @return 最小生成树的边列表
     */
    public static List<Edge> primMST(int[][] graph) {
        int V = graph.length;
        List<Edge> mst = new ArrayList<>();
        
        // 记录顶点是否在MST中
        boolean[] inMST = new boolean[V];
        
        // 记录顶点到MST的最小权重
        int[] key = new int[V];
        Arrays.fill(key, Integer.MAX_VALUE);
        
        // 记录顶点的父节点
        int[] parent = new int[V];
        Arrays.fill(parent, -1);
        
        // 从顶点0开始
        key[0] = 0;
        
        // 构建包含V-1条边的MST
        for (int count = 0; count < V - 1; count++) {
            // 选择不在MST中且key值最小的顶点
            int u = minKey(key, inMST);
            inMST[u] = true;
            
            // 更新相邻顶点的key值
            for (int v = 0; v < V; v++) {
                if (graph[u][v] != 0 && !inMST[v] && graph[u][v] < key[v]) {
                    parent[v] = u;
                    key[v] = graph[u][v];
                }
            }
        }
        
        // 构建MST边列表
        for (int i = 1; i < V; i++) {
            if (parent[i] != -1) {
                mst.add(new Edge(parent[i], i, graph[i][parent[i]]));
            }
        }
        
        return mst;
    }
    
    private static int minKey(int[] key, boolean[] inMST) {
        int min = Integer.MAX_VALUE;
        int minIndex = -1;
        
        for (int v = 0; v < key.length; v++) {
            if (!inMST[v] && key[v] < min) {
                min = key[v];
                minIndex = v;
            }
        }
        return minIndex;
    }
}
```

### 方法二：使用优先队列（适合稀疏图）

```java
import java.util.*;

public class PrimWithPQ {
    
    static class Edge implements Comparable<Edge> {
        int from, to, weight;
        
        public Edge(int from, int to, int weight) {
            this.from = from;
            this.to = to;
            this.weight = weight;
        }
        
        @Override
        public int compareTo(Edge other) {
            return Integer.compare(this.weight, other.weight);
        }
    }
    
    /**
     * 使用优先队列实现Prim算法
     * @param adjList 邻接表表示的图
     * @param V 顶点数量
     * @return 最小生成树的边列表和总权重
     */
    public static PrimResult primAdjacencyList(List<List<Edge>> adjList, int V) {
        List<Edge> mst = new ArrayList<>();
        boolean[] visited = new boolean[V];
        PriorityQueue<Edge> pq = new PriorityQueue<>();
        int totalWeight = 0;
        
        // 从顶点0开始
        visited[0] = true;
        pq.addAll(adjList.get(0));
        
        while (!pq.isEmpty() && mst.size() < V - 1) {
            Edge edge = pq.poll();
            
            // 如果两个顶点都在MST中，跳过
            if (visited[edge.to]) {
                continue;
            }
            
            // 添加边到MST
            mst.add(edge);
            totalWeight += edge.weight;
            visited[edge.to] = true;
            
            // 添加新顶点的所有边到优先队列
            for (Edge nextEdge : adjList.get(edge.to)) {
                if (!visited[nextEdge.to]) {
                    pq.offer(nextEdge);
                }
            }
        }
        
        return new PrimResult(mst, totalWeight);
    }
    
    static class PrimResult {
        List<Edge> mst;
        int totalWeight;
        
        public PrimResult(List<Edge> mst, int totalWeight) {
            this.mst = mst;
            this.totalWeight = totalWeight;
        }
    }
}
```

### 完整的测试示例

```java
public class PrimTest {
    public static void main(String[] args) {
        // 示例图
        int V = 5;
        int[][] graph = {
            {0, 2, 0, 6, 0},
            {2, 0, 3, 8, 5},
            {0, 3, 0, 0, 7},
            {6, 8, 0, 0, 9},
            {0, 5, 7, 9, 0}
        };
        
        // 测试邻接矩阵版本
        System.out.println("使用邻接矩阵的Prim算法:");
        List<Edge> mst1 = PrimAlgorithm.primMST(graph);
        printMST(mst1);
        
        // 测试邻接表版本
        System.out.println("\n使用邻接表的Prim算法:");
        List<List<Edge>> adjList = createAdjacencyList(graph);
        PrimWithPQ.PrimResult result = PrimWithPQ.primAdjacencyList(adjList, V);
        printMST(result.mst);
        System.out.println("总权重: " + result.totalWeight);
    }
    
    private static List<List<Edge>> createAdjacencyList(int[][] graph) {
        int V = graph.length;
        List<List<Edge>> adjList = new ArrayList<>();
        
        for (int i = 0; i < V; i++) {
            adjList.add(new ArrayList<>());
            for (int j = 0; j < V; j++) {
                if (graph[i][j] != 0) {
                    adjList.get(i).add(new Edge(i, j, graph[i][j]));
                }
            }
        }
        return adjList;
    }
    
    private static void printMST(List<Edge> mst) {
        System.out.println("最小生成树的边:");
        for (Edge edge : mst) {
            System.out.println(edge.from + " - " + edge.to + " : " + edge.weight);
        }
    }
}
```

## 边类定义

```java
class Edge {
    int from, to, weight;
    
    public Edge(int from, int to, int weight) {
        this.from = from;
        this.to = to;
        this.weight = weight;
    }
    
    @Override
    public String toString() {
        return from + " - " + to + " : " + weight;
    }
}
```

## 算法特点

### 优点：
1. **保证最优解**：总能找到最小生成树
2. **实现简单**：逻辑清晰，易于理解和实现
3. **适用于稠密图**：在稠密图中表现良好

### 缺点：
1. **性能依赖数据结构**：不同的实现方式性能差异较大
2. **不适合动态图**：图结构变化时需要重新计算

## 应用场景

1. 网络设计（电缆布线、网络连接）
2. 电路设计
3. 交通规划
4. 集群分析
5. 图像分割

Prim 算法是解决最小生成树问题的经典算法，理解其原理和实现对于掌握图算法非常重要。
